## 🔧 Если Что-то Не Работает

### MiniApp не обновляется:
```bash
# Жесткая перезагрузка
Windows: Ctrl + Shift + R
Mac: Cmd + Shift + R

# Или очисти кэш браузера
Chrome: F12 → Application → Clear storage
```

### Бот не отвечает:
```bash
# Переустанови webhook
node scripts/setup-webhook-production.js

# Проверь логи в Vercel
https://vercel.com/dashboard → Logs
```

### Footer не отображается:
```bash
# Проверь в консоли (F12)
console.log(document.querySelector('.app-footer'));

# Должен вернуть: <footer class="app-footer">...</footer>
```
