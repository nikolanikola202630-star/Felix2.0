## 📋 ЧТО НУЖНО СДЕЛАТЬ

### 1. Добавить свой ID в администраторы:

```javascript
// lib/admin/admin-system.js
const ADMIN_IDS = [
  123456, // Замени на свой Telegram ID
];
```

**Как узнать свой ID:**
1. Открой бота @userinfobot в Telegram
2. Отправь ему любое сообщение
3. Он пришлет твой ID
4. Добавь этот ID в массив ADMIN_IDS

### 2. Задеплоить с твоим ID:

```bash
# Отредактируй lib/admin/admin-system.js
# Добавь свой ID

git add lib/admin/admin-system.js
git commit -m "Add admin ID"
git push
vercel --prod
```

### 3. Протестировать:

1. **Открой Mini App:**
   - https://felix2-0.vercel.app/miniapp/elite.html

2. **Проверь визуал:**
   - Все карточки должны быть красивыми
   - Hover эффекты работают
   - Анимации плавные

3. **Проверь админ-функции:**
   - Попробуй подать заявку на партнерство
   - Проверь что заявка создалась

4. **Добавь первый курс (как админ):**
```javascript
// Через консоль браузера в Mini App
await fetch('/api/admin', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    action: 'addCourse',
    userId: YOUR_ADMIN_ID, // Твой ID
    data: {
      title: 'JavaScript Основы',
      description: 'Полный курс по JavaScript для начинающих',
      icon: '💻',
      lessons_count: 24,
      duration: '8 часов',
      rating: 4.8,
      students_count: 0
    }
  })
});
```
