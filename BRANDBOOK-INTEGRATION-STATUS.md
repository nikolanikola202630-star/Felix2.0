## 💡 Рекомендации

### Для разработчиков
1. Используйте только классы с суффиксом `-brandbook`
2. Следуйте структуре компонентов из `brandbook-components.css`
3. Используйте переменные из `brandbook-variables.css`
4. Не создавайте новые цвета или шрифты

### Для дизайнеров
1. Золото только для акцентов (≤ 10%)
2. Проверяйте контрастность цветов
3. Используйте только утверждённые шрифты
4. Следуйте визуальной иерархии

### Для копирайтеров
1. Обращение только на "Вы"
2. Предложения до 15 слов
3. Без смайликов (кроме 🦁 ⚜️ 📜)
4. Без восклицательных знаков

### Для менеджеров
1. Проверяйте соответствие брендбуку
2. Контролируйте сроки
3. Собирайте обратную связь
4. Координируйте команду
