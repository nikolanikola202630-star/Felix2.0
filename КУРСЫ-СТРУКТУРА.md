## 💾 Техническая реализация:

### Файлы:
- `data/courses-structure.json` - Полная структура курсов
- `api/courses-full.js` - API endpoint для получения курсов
- `bot.js` - Бот использует структуру из JSON
- `miniapp/js/app.js` - Mini App загружает курсы через API

### API Endpoints:
```
GET /api/courses-full
GET /api/courses-full?id=1
GET /api/courses-full?category=psychology
GET /api/courses-full?level=beginner
GET /api/courses-full?free_only=true
```

### Ответ API:
```json
{
  "success": true,
  "count": 3,
  "courses": [...]
}
```
