## 🔧 Настройка

### 1. Vercel KV (Redis)

1. Перейти в Vercel Dashboard → Storage → Create Database → KV
2. Скопировать переменные окружения:
   ```
   KV_URL=...
   KV_REST_API_URL=...
   KV_REST_API_TOKEN=...
   KV_REST_API_READ_ONLY_TOKEN=...
   ```
3. Добавить в Vercel Environment Variables

### 2. Sentry

1. Создать проект на https://sentry.io
2. Скопировать DSN
3. Добавить в Environment Variables:
   ```
   SENTRY_DSN=https://...@sentry.io/...
   ```

### 3. AI Rate Limits

Добавить в Environment Variables:
```
AI_DAILY_LIMIT=50
AI_HOURLY_LIMIT=10
```

## 📝 Использование новых модулей

### Кэширование

```javascript
import { cache, cacheKeys } from './lib/cache.js';

// Получить с кэшированием
const settings = await cache.get(
  cacheKeys.userSettings(userId),
  () => db.getUserSettings(userId),
  3600 // TTL 1 час
);

// Инвалидировать
await cache.invalidate(cacheKeys.userSettings(userId));
```

### Мониторинг

```javascript
import { initMonitoring, captureError, logger } from './lib/monitoring.js';

// Инициализация (в начале приложения)
initMonitoring();

// Логирование
logger.info('User logged in', { userId });

// Отслеживание ошибок
try {
  await processMessage();
} catch (error) {
  captureError(error, { userId, chatId });
}
```

### AI Rate Limiting

```javascript
import { checkAILimit, incrementAIUsage } from './lib/ai-rate-limit.js';

// Проверить лимит
const limit = await checkAILimit(userId);
if (!limit.allowed) {
  return `Лимит исчерпан (${limit.limit}). Попробуйте через ${formatTime(limit.resetAt)}`;
}

// Инкрементировать после использования
await incrementAIUsage(userId, tokens);
```

### Graceful Shutdown

```javascript
import { setupGracefulShutdown } from './lib/shutdown.js';

// В начале приложения
setupGracefulShutdown();
```

## 🔄 Интеграция в существующий код

### 1. Обновить api/webhook.js

```javascript
import { initMonitoring, captureError, logger } from '../lib/monitoring.js';
import { cache, cacheKeys } from '../lib/cache.js';
import { checkAILimit, incrementAIUsage } from '../lib/ai-rate-limit.js';

// В начале файла
initMonitoring();

// В обработчике
export default async function handler(req, res) {
  try {
    // Проверить AI лимит
    const limit = await checkAILimit(userId);
    if (!limit.allowed) {
      return res.json({ text: `Лимит AI запросов исчерпан (${limit.limit}/день)` });
    }

    // Получить настройки с кэшированием
    const settings = await cache.get(
      cacheKeys.userSettings(userId),
      () => db.getUserSettings(userId),
      3600
    );

    // AI запрос
    const response = await ai.getChatResponse(message, history, settings);

    // Инкрементировать счетчик
    await incrementAIUsage(userId, response.tokens);

    // Логирование
    logger.info('Message processed', {
      userId,
      tokens: response.tokens,
      latency: response.latency
    });

    res.json({ text: response.content });
  } catch (error) {
    captureError(error, { userId, chatId });
    res.status(500).json({ error: 'Internal error' });
  }
}
```

### 2. Обновить api/settings.js
