## Архитектура V12

### 1. Единая система навигации

- Breadcrumbs
- Единый стиль навигации

### 2. Модульная структура
```
miniapp/
├── core/                    # Ядро системы
│   ├── router.js           # Роутинг
│   ├── state.js            # Глобальное состояние
│   ├── api.js              # API клиент
│   └── utils.js            # Утилиты
├── components/             # Переиспользуемые компоненты
│   ├── navigation.js
│   ├── cards.js
│   ├── modals.js
│   └── forms.js
├── pages/                  # Страницы
│   ├── home/
│   ├── academy/
│   ├── settings/
│   ├── community/
│   └── profile/
└── styles/                 # Стили
    ├── variables.css       # CSS переменные
    ├── components.css      # Компоненты
    └── pages.css           # Страницы
```

### 3. Единый дизайн-система
- Цветовая палитра
- Типографика
- Компоненты UI
- Анимации
- Иконки

### 4. Оптимизация производительности
- Lazy loading

### 5. Улучшенный UX
