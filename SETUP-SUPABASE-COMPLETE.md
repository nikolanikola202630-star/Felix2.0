## 📋 Что такое Supabase?

Supabase - это open-source альтернатива Firebase с PostgreSQL базой данных.

### Преимущества:
- ✅ Полноценная PostgreSQL база данных
- ✅ Full-text search
- ✅ Real-time subscriptions
- ✅ Storage для файлов
- ✅ Бесплатный tier (500 MB, 2 GB bandwidth)
- ✅ Автоматические backups
- ✅ REST API из коробки
