## 🔗 Реферальная система

### Как работает:

1. **Партнер получает ссылку:**
   ```
   https://t.me/AcademyBot?start=ref_partner123
   ```

2. **Пользователь кликает:**
   - Открывается бот
   - Бот отправляет приветствие
   - Кнопка "Открыть Академию"

3. **Mini App получает код:**
   ```javascript
   const startParam = tg.initDataUnsafe.start_param; // "ref_partner123"
   ```

4. **Сохранение в БД:**
   ```javascript
   // TODO: Реализовать
   await saveReferral(user.id, startParam);
   ```

5. **При покупке:**
   - Партнеру начисляется 20%
   - Рефереру начисляется 10% бонусов
