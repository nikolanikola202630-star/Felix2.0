### Критическое (1-2 дня)

**1. Настроить Supabase (30 мин)**
```bash
# Установить зависимости
npm install

# Настроить .env
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_KEY=eyJ...

# Применить SQL функцию
# Скопировать database/functions/update_bonus_balance.sql
# Выполнить в Supabase Dashboard

# Протестировать
npm run test:supabase
```

**2. Настроить Telegram Stars (1 час)**
- Открыть @BotFather
- Выбрать бота Felix Academy
- Payments → Connect payment provider
- Выбрать Telegram Stars
- Протестировать покупку

**3. Добавить переменные в Vercel (10 мин)**
```bash
vercel env add SUPABASE_URL production
vercel env add SUPABASE_KEY production
```

**4. Задеплоить (5 мин)**
```bash
vercel --prod
```

**5. Протестировать (1 час)**
- Health check
- Покупка курса
- Начисление комиссий
- Проверка доступа
