## 🎯 Обзор проекта

**Felix Academy** - это комплексная платформа онлайн-образования с:
- Telegram Mini App интерфейсом
- Админ панелью для управления
- Партнерской системой
- AI ассистентом
- Системой автоматизации
- Реферальной программой

**Версия:** V12.0 (Глобальное обновление)
**Статус:** Production Ready
**Технологии:** Node.js, Supabase, Vercel, Telegram Bot API

## 🤖 Автоматизация V12

### Компоненты

#### 1. Auto Sync
- **Функция:** Синхронизация данных
- **Интервал:** Каждые 5 минут
- **Задачи:**
  - Синхронизация пользователей
  - Синхронизация курсов
  - Синхронизация статистики
  - Очистка кэша
  - Проверка версий файлов

#### 2. Health Monitor
- **Функция:** Мониторинг здоровья
- **Интервал:** Каждые 2 минуты
- **Проверки:**
  - Доступность эндпоинтов
  - Подключение к БД
  - Использование памяти
  - Свободное место
  - Генерация алертов

#### 3. Auto Deploy
- **Функция:** Автоматический деплой
- **Интервал:** Мин. 5 минут между деплоями
- **Процесс:**
  - Отслеживание изменений в Git
  - Оптимизация кода
  - Запуск тестов
  - Git commit и push
  - Ожидание Vercel деплоя

#### 4. Scheduled Tasks
- **Ежедневные (03:00):**
  - Очистка старых логов
  - Очистка health checks
  - Очистка кэша
  
- **Еженедельные (Вс 20:00):**
  - Генерация отчета
  - Отправка администраторам
  
- **Ежемесячные (1-е число 02:00):**
  - Оптимизация БД
  - Полная очистка кэша
  - Месячный отчет

## 🚀 Деплой

### Платформы

**Production:**
- Vercel (Frontend + API)
- Supabase (Database)
- GitHub (Code repository)

**Staging:**
- Локальная разработка
- Тестовая БД

### Процесс деплоя

```
1. Разработка локально
2. Git commit
3. Git push origin main
4. Vercel автоматически деплоит
5. Проверка через check-deploy-v12.js
6. Мониторинг через Health Monitor
```

### Время деплоя

- HTML файлы: ~30 сек
- CSS/JS файлы: ~2-3 мин
- API endpoints: ~3-5 мин
