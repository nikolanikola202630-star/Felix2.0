## Если проблема сохраняется

### Вариант 1: Очистить кэш Telegram

**На телефоне:**
1. Настройки → Данные и память → Использование памяти
2. Очистить кэш
3. Перезапустить Telegram

**На компьютере:**
1. Закрыть Telegram полностью
2. Удалить папку кэша:
   - Windows: `%APPDATA%\Telegram Desktop\tdata\user_data`
   - Mac: `~/Library/Application Support/Telegram Desktop/tdata`
   - Linux: `~/.local/share/TelegramDesktop/tdata`
3. Запустить Telegram

### Вариант 2: Использовать Telegram Web

Откройте: https://web.telegram.org

Telegram Web не кэширует Mini Apps так агрессивно.

### Вариант 3: Проверить переменные Vercel

1. Откройте: https://vercel.com/dashboard
2. Выберите проект: `felix2-0`
3. Settings → Environment Variables
4. Проверьте:
   ```
   MINIAPP_URL=https://felix2-0.vercel.app/miniapp/egoist.html
   ```
5. Если неправильно - исправьте и сделайте Redeploy

### Вариант 4: Ручной редеплой

```bash
vercel --prod
```

Или через Dashboard:
1. Deployments → Latest
2. ... → Redeploy
