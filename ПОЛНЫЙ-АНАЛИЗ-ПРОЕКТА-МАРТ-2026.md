## 🎯 ОБЗОР ПРОЕКТА

Felix Academy - комплексная образовательная платформа на базе Telegram:
- 🤖 Telegram бот с AI (Groq Llama 3.3 70B)
- 📱 Mini App с современным UI
- 🎓 Система онлайн-курсов (3 курса, 55 уроков, 70 часов)
- 💼 Партнерская система для создателей контента
- ⚙️ Админ-панели для управления
- 💳 Интеграция платежей (Telegram Stars)
- 👥 Сообщество и персонализация

**Стек:** Node.js + Vercel Serverless, PostgreSQL (Supabase), Telegram Bot API, Groq AI
