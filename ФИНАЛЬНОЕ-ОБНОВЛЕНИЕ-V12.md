## 🤖 Автоматизация

### Auto Sync V12
**Функции:**
- Синхронизация пользователей
- Синхронизация курсов
- Синхронизация статистики
- Очистка кэша
- Проверка версий файлов

**Интервал:** Каждые 5 минут

**Файл:** `lib/automation/auto-sync-v12.js`

### Health Monitor V12
**Проверки:**
- Доступность эндпоинтов
- Подключение к БД
- Использование памяти
- Свободное место на диске

**Интервал:** Каждые 2 минуты

**Статусы:** healthy, degraded, critical, down

**Файл:** `lib/automation/health-monitor-v12.js`

### Auto Deploy V12
**Функции:**
- Отслеживание изменений в Git
- Автоматическая оптимизация
- Запуск тестов
- Git commit и push
- Ожидание Vercel деплоя

**Интервал:** Минимум 5 минут между деплоями

**Файл:** `lib/automation/auto-deploy-v12.js`

### Master Automation V12
**Управление:**
- Запуск всех подсистем
- Координация работы
- Запланированные задачи
- Генерация отчетов
- Логирование

**Файл:** `lib/automation/master-automation-v12.js`

## 🚀 Запуск

### Локально

```bash
# Установка зависимостей
npm install

# Запуск приложения
npm start

# Запуск автоматизации
node scripts/start-automation-v12.js

# Оптимизация
node scripts/optimize-project-v12.js

# Синхронизация
node scripts/sync-all-v12.js

# Деплой
node scripts/deploy-v12.js
```

### Production

```bash
# Деплой на Vercel
git push origin main

# Ожидание 2-3 минуты
# Vercel автоматически задеплоит изменения
```
