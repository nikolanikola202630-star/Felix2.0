### 🤖 БОТ (Telegram)

**Основной функционал:**
- ✅ AI-ассистент с Groq (llama-3.3-70b-versatile)
- ✅ Интеграция с PostgreSQL через Supabase
- ✅ Rate limiting: 50 запросов/день на пользователя
- ✅ История сообщений с контекстом
- ✅ Автосохранение пользователей в БД
- ✅ Логирование ошибок

**Команды:**
- `/start` - Главное меню с EGOIST брендингом
- `/help` - Справка по командам

- `/profile` - Профиль с статистикой из БД
- `/partner` - Информация о партнерке
- `/partner_panel` - Открыть партнерский кабинет
- `/admin` - Админ-панель (только для админов)
- **Любое сообщение** - AI отвечает автоматически

**База данных:**
- Сохранение пользователей
- История сообщений
- Статистика использования AI
- Прогресс по курсам
- Достижения
