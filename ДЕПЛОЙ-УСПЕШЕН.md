## 🚀 Следующие шаги

### 1. Настроить Telegram Stars (5 мин)

1. Открыть @BotFather
2. Выбрать бота: @felix_academy_bot
3. Payments → Connect payment provider
4. Выбрать: Telegram Stars
5. Подтвердить

### 2. Загрузить курсы (10 мин)

1. Открыть админ панель:
   ```
   https://felix2-0.vercel.app/miniapp/admin-courses.html
   ```

2. Импортировать файл:
   ```
   data/courses-structure.json
   ```

3. Проверить, что курсы загружены

### 3. Протестировать покупку (5 мин)

1. Открыть Mini App в Telegram
2. Выбрать любой курс
3. Нажать "Купить"
4. Оплатить через Telegram Stars
5. Проверить в БД:
   ```sql
   SELECT * FROM purchases ORDER BY created_at DESC LIMIT 5;
   ```

### 4. Пригласить beta-тестеров (1 день)

1. Создать группу в Telegram
2. Пригласить 10-20 человек
3. Дать доступ к Mini App
4. Собрать feedback
