# 🚀 Голосовой помощник - Быстрый старт

## ⚡ За 5 минут

### 1. Установите зависимости

```bash
npm install
```

Новые пакеты:
- `express` - веб-сервер
- `multer` - загрузка файлов
- `form-data` - работа с формами

### 2. Получите Groq API ключ

1. Откройте [console.groq.com](https://console.groq.com)
2. Войдите через Google/GitHub
3. Settings → API Keys → Create Key
4. Скопируйте ключ (начинается с `gsk_`)

### 3. Добавьте в .env

```env
GROQ_API_KEY=gsk_ваш_ключ_здесь
```

### 4. Протестируйте

```bash
npm run test:voice
```

Должно пройти 2 теста:
- ✅ LLM (генерация текста)
- ✅ TTS (озвучивание)

### 5. Запустите

Локально:
```bash
vercel dev
```

Или деплой:
```bash
vercel --prod
```

### 6. Откройте Mini App

В Telegram Mini App нажмите кнопку **🎙️ Голос**

Или откройте напрямую:
```
https://your-domain.com/miniapp/voice-assistant.html
```

## 🎯 Как использовать

1. **Удерживайте** кнопку микрофона
2. **Говорите** свой вопрос
3. **Отпустите** кнопку
4. Получите **голосовой ответ**

## 🐛 Проблемы?

### Ошибка "No audio file"
- Проверьте доступ к микрофону в браузере
- Убедитесь, что используете HTTPS

### Ошибка "GROQ_API_KEY not found"
- Проверьте файл `.env`
- Перезапустите сервер после добавления ключа

### Ошибка "Invalid API key"
- Проверьте, что ключ начинается с `gsk_`
- Создайте новый ключ на console.groq.com
