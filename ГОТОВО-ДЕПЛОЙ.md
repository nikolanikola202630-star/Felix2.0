## 🔐 СПОСОБ 2: Personal Access Token

### Автоматический скрипт
```powershell
.\push-with-token.ps1
```

Скрипт:
1. Попросит ввести токен
2. Настроит remote
3. Выполнит push
4. Удалит токен из конфигурации (безопасность)

### Как получить токен
1. Перейти: https://github.com/settings/tokens
2. Generate new token (classic)
3. Выбрать: ✅ repo (full control)
4. Generate token
5. **СКОПИРОВАТЬ ТОКЕН** (показывается один раз!)

### Ручной push с токеном
```powershell
git remote set-url origin https://YOUR_TOKEN@github.com/nikolanikola202630-star/Felix2.0.git
git push -u origin main
git remote set-url origin https://github.com/nikolanikola202630-star/Felix2.0.git
```

## 🔑 СПОСОБ 4: SSH ключ

### Создание ключа
```powershell
ssh-keygen -t ed25519 -C "your_email@example.com"
```

### Добавление на GitHub
1. Скопировать публичный ключ:
   ```powershell
   cat ~/.ssh/id_ed25519.pub
   ```
2. GitHub → Settings → SSH and GPG keys → New SSH key
3. Вставить ключ

### Push
```powershell
git remote set-url origin git@github.com:nikolanikola202630-star/Felix2.0.git
git push -u origin main
```
