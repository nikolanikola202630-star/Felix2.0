## 🔗 КАК РАБОТАЕТ

### 1. Партнер получает ссылку:
```
https://t.me/felix_ref_bot?start=ref_123456
```
Где `123456` - ID партнера

### 2. Пользователь переходит по ссылке:
- Открывается реферальный бот
- Бот приветствует и показывает, от кого ссылка
- Клик сохраняется в БД

### 3. Защита от накрутки:
- Проверка IP (max 5 кликов/час с одного IP)
- Проверка Session ID
- Блокировка подозрительных кликов
- Все сохраняется в `referral_clicks`

### 4. Перенаправление:
- Кнопка "Открыть Академию" → MiniApp
- Кнопка "Перейти в основного бота" → @fel12x_bot
- Кнопка "Стать партнером"
