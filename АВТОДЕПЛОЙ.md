## ⚡ Быстрый деплой через Vercel CLI

### Шаг 1: Установить Vercel CLI (если еще не установлен)

```powershell
npm install -g vercel
```

### Шаг 2: Войти в Vercel

```powershell
vercel login
```

Откроется браузер для авторизации.

### Шаг 3: Деплой с переменными окружения

```powershell
# Установить переменные
vercel env add TELEGRAM_BOT_TOKEN production
# Вставить: 8623255560:AAE7sC-7-eWA5LD-ebATDUh6nGUG0pYm03U

vercel env add GROQ_API_KEY production
# Вставить: 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

vercel env add DATABASE_URL production
# Вставить: postgresql://postgres:[YOUR-PASSWORD]@db.nnegvsxhspdnsvjpcscn.supabase.co:5432/postgres

vercel env add ADMIN_ID production
# Вставить: 8264612178

vercel env add MINIAPP_URL production
# Вставить: https://your-project.vercel.app/miniapp/elite.html

vercel env add NODE_ENV production
# Вставить: production

# Деплой
vercel --prod
```

## 🌐 Или через Vercel Dashboard (Проще!)

https://vercel.com/new

- Выберите **Felix2.0**

⚠️ **Замените [YOUR-PASSWORD] на пароль от Supabase!**

### 5. Deploy
Нажмите **Deploy** и дождитесь завершения (~2 мин)

## 📋 После деплоя

### 1. Скопируйте URL проекта
Например: `https://felix2-0-abc123.vercel.app`

### 2. Обновите MINIAPP_URL
```powershell
vercel env rm MINIAPP_URL production
vercel env add MINIAPP_URL production
# Вставить: https://[ваш-url].vercel.app/miniapp/elite.html

vercel --prod
```

### 3. Установите Webhook
```powershell
$token = "8623255560:AAE7sC-7-eWA5LD-ebATDUh6nGUG0pYm03U"
$url = "https://[ваш-url].vercel.app/api/webhook"

Invoke-RestMethod -Uri "https://api.telegram.org/bot$token/setWebhook" -Method Post -Body @{url=$url}
```

### 4. Проверьте
```powershell
# Health check
Invoke-RestMethod -Uri "https://[ваш-url].vercel.app/api/webhook"

# Webhook info
Invoke-RestMethod -Uri "https://api.telegram.org/bot$token/getWebhookInfo"
```
