## 🚀 Что нужно для запуска

### 1. Обновить URL в боте (5 минут)

**Файл:** `bot.js` или `bot-v12.js`

```javascript
// Изменить
const MINIAPP_URL = 'https://felix2-0.vercel.app/miniapp/egoist.html';
```

**Или в .env:**
```env
MINIAPP_URL=https://felix2-0.vercel.app/miniapp/egoist.html
```

### 2. Создать Telegram-группы (10 минут)

Создать 5 групп:
1. EGOIST ACADEMY KINGS
2. EGOIST BRAIN
3. EGOIST IT
4. EGOIST TRADING
5. EGOIST SUCCESS

Получить ссылки-приглашения и обновить в:
- `miniapp/js/egoist-app.js` (метод renderCommunity)
- `data/egoist-courses.json` (поле community_link)

### 3. Добавить реальный контент (опционально)

**Сейчас:**
- 5 категорий с placeholder данными
- Тестовые YouTube ссылки
- Placeholder PDF ссылки

**Для production:**
- Записать видео
- Загрузить на YouTube
- Создать PDF материалы
- Обновить ссылки в `data/egoist-courses.json`

### 4. Деплой (5 минут)

```bash
# Commit
git add .
git commit -m "feat: добавить EGOIST ACADEMY минималистичный дизайн"
git push

# Vercel автоматически задеплоит
```

### 5. Обновить webhook (2 минуты)

```bash
# Если нужно
curl -X POST "https://api.telegram.org/bot<TOKEN>/setWebhook" \
  -d "url=https://felix2-0.vercel.app/api/webhook"
```
