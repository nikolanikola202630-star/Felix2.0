## 📝 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### API Endpoints обновлены:

```javascript
// Теперь используются реальные API:
GET  /api/stats?user_id={id}           // Статистика пользователя
POST /api/learning                      // AI рекомендации
GET  /api/history?user_id={id}         // История активности
POST /api/partner                       // Реферальная система
POST /api/payments?action=create-invoice // Создание invoice
POST /api/payments?action=webhook       // Обработка платежей
```

### Fallback стратегия:

Все функции имеют fallback на тестовые данные при ошибках API:
- Статистика → нули
- Курсы → placeholder данные
- Активность → пустой список
- AI рекомендации → общие советы

Это обеспечивает работоспособность даже при проблемах с API.
