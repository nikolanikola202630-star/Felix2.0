## 📝 Пошаговая инструкция (Вариант 1)

### Шаг 1: Подготовка (5 минут)

```bash
# 1.1. Убедиться, что все изменения закоммичены
git status

# 1.2. Создать резервную копию
git add .
git commit -m "Backup before brandbook integration"

# 1.3. Создать ветку для брендбука (опционально)
```

```bash

# Скрипт автоматически:
# - Создаст резервные копии в miniapp/backup-pre-brandbook/
# - Заменит старые файлы новыми
# - Обновит ссылки
# - Сгенерирует отчёт BRANDBOOK-INTEGRATION-REPORT.json
```

### Шаг 3: Проверка (10 минут)

```bash
# 3.1. Открыть в браузере
# miniapp/elite.html
```

## 🔄 Откат изменений

### Если что-то пошло не так

#### Вариант 1: Git откат

```bash
# Откатить последний коммит
git reset --hard HEAD~1

# Или откатить к конкретному коммиту
git log  # найти хеш коммита
git reset --hard <commit-hash>
```

#### Вариант 2: Восстановление из бэкапа

```bash
# Восстановить файлы из miniapp/backup-pre-brandbook/
cp miniapp/backup-pre-brandbook/elite.html miniapp/elite.html
```

## 🐛 Возможные проблемы и решения

### Проблема 1: Шрифты не загружаются

**Решение:**
```html
<!-- Добавить preconnect в <head> -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
```

### Проблема 2: Цвета отображаются неправильно

**Решение:**
```bash
# Проверить, что подключён brandbook-variables.css
# Проверить порядок подключения CSS файлов
```

### Проблема 3: Анимации тормозят

**Решение:**
```css
/* Использовать transform вместо left/top */
/* Использовать will-change для оптимизации */
.element {
  will-change: transform;
  transform: translateX(0);
}
```

**Решение:**
