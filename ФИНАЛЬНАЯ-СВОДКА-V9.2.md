- ✅ Ссылки на политику конфиденциальности

#### Технические детали
```javascript
// localStorage структура
{
  "felixSettings": {
    "notifications": true,
    "course-updates": true,
    "achievements": true,
    "autoplay": false,
    "offline": false
  }
}
```

#### Класс SettingsManager
- `loadSettings()` - Загрузка из localStorage
- `saveSettings()` - Сохранение в localStorage
- `toggleSetting(key)` - Переключение настройки
- `clearCache()` - Очистка кеша с подтверждением
- `checkUpdates()` - Проверка обновлений
- `hapticFeedback(type)` - Тактильная обратная связь
