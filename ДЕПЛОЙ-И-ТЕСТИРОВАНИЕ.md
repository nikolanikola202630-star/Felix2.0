## 🚀 Шаги для деплоя

### 1. Проверка переменных окружения
```bash
# Проверить наличие всех необходимых переменных
cat .env
```

Необходимые переменные:
- `BOT_TOKEN` - Telegram Bot Token
- `SUPABASE_URL` - URL Supabase
- `SUPABASE_KEY` - Anon Key Supabase
- `GROQ_API_KEY` - API ключ Groq
- `ADMIN_IDS` - ID администраторов

### 2. Коммит изменений
```bash
git add .
git commit -m "feat: добавлен каталог, поиск, обновлены community и settings"
git push origin main
```

### 3. Деплой на Vercel
```bash
# Автоматический деплой через GitHub
# или
vercel --prod
```

### 4. Установка webhook
```bash
node scripts/set-webhook.js
```
