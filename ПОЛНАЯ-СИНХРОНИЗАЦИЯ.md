## 💻 Команды для запуска:

```bash
# 1. Установить зависимости
npm install

# 2. Запустить локальный сервер (для Mini App)
node server-local.js

# 3. Запустить бота (в другом терминале)
node bot.js

# 4. Открыть в браузере
http://localhost:3000/miniapp/index.html
```
