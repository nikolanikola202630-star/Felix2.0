# ✅ Каталог и поиск готовы - Felix Academy

## 🎯 Что сделано

### 1. Catalog.html - Полноценный каталог курсов
**Файлы:**
- `miniapp/catalog.html` - HTML страница
- `miniapp/js/catalog.js` - Модуль управления каталогом

**Функционал:**
- ✅ Фильтры по категориям (Психология, Саморазвитие, Бизнес)
- ✅ Фильтры по уровню (Начальный, Средний, Продвинутый)
- ✅ Поиск в реальном времени (с debounce 300ms)
- ✅ Сортировка:
  - По популярности (количество студентов)
  - По рейтингу
  - По цене (возрастание/убывание)
  - По новизне
- ✅ Карточки курсов с информацией:
  - Изображение курса
  - Категория и уровень
  - Название и описание
  - Рейтинг, длительность, количество уроков
  - Цена и количество студентов
- ✅ Skeleton loaders при загрузке
- ✅ Empty state при отсутствии результатов
- ✅ Haptic feedback для Telegram
- ✅ Responsive дизайн
- ✅ Интеграция с API (с fallback на локальные данные)

**Технические особенности:**
- Класс `CatalogManager` для управления состоянием
- Debounce для оптимизации поиска
- XSS защита через `escapeHtml()`
- Lazy loading изображений
- Оптимизированная фильтрация и сортировка

### 2. Search.html - Глобальный поиск
**Файл:** `miniapp/search.html`

**Функционал:**

- ✅ Популярные запросы (быстрый доступ)
- ✅ Кнопка очистки поиска
- ✅ Автофокус на поле ввода
- ✅ Счетчик результатов
- ✅ Empty state при отсутствии результатов
- ✅ Удаление из истории поиска

**Технические особенности:**
- Класс `SearchManager` для управления поиском
- Индексация всех данных для быстрого поиска
- Подготовка данных из структуры курсов
- Highlight совпадений в результатах
- Pluralization для русского языка
- Debounce для оптимизации

## 📊 Интеграция с данными

### API Endpoints
```javascript
GET /api/courses - Список всех курсов
```

### Fallback
При недоступности API используются локальные данные:
```
../data/courses-structure.json
```

### Структура данных
```javascript
{
  courses: [
    {
      id, slug, title, category, description,
      price, rating, students, duration_hours,
      level, image, instructor,
      themes: [
        {
          id, title, file,
          lessons: [...]
        }
      ]
    }
  ]
}
```

## 🚀 Навигация

### Catalog.html
- Клик на карточку → `course.html?slug={slug}`
- Bottom navigation для переходов

### Search.html
- Клик на курс → `course.html?slug={slug}`
- Клик на урок → `lesson.html?course={slug}&lesson={id}`
- Клик на преподавателя → `course.html?slug={slug}`

## 📱 Telegram WebApp интеграция

### Используемые фичи
- `tg.ready()` - инициализация
- `tg.expand()` - полноэкранный режим
- `tg.HapticFeedback` - тактильная обратная связь
  - `light` - легкое касание (фильтры, поиск)
  - `medium` - среднее касание (открытие курса)
