## Если не работает

### Проблема: Старая версия открывается

**Причина:** Кэш Telegram

**Решение:**
```bash
# Уже выполнено
node scripts/clear-telegram-cache.js
```

Дополнительно:
1. Закрыть бота полностью
2. Подождать 2-3 минуты
3. Открыть снова

Или:
1. Telegram → Настройки → Данные и память
2. Очистить кэш
3. Перезапустить Telegram

### Проблема: Черный экран

**Причина:** JS не загрузился

**Решение:**
1. Проверить консоль браузера (F12)
2. Проверить, что все файлы доступны:
   - https://felix2-0.vercel.app/miniapp/egoist.html
   - https://felix2-0.vercel.app/miniapp/css/egoist-theme.css
   - https://felix2-0.vercel.app/miniapp/js/egoist-app.js

### Проблема: Каталог не загружается

**Причина:** Данные курсов недоступны

**Решение:**
Встроить данные в `egoist-catalog.html`:

```javascript
// В начале файла
const COURSES_DATA = {
  "courses": [
    // ... данные из data/egoist-courses.json
  ]
};
```

```javascript

  // Заменить 'https://t.me/+...' на реальные ссылки
}
```

1. Записать видео → YouTube
2. Создать PDF → загрузить
