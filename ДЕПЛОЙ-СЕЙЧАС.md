### Шаг 1: Создать базу данных Supabase (10 мин)

1. **Открыть:** https://supabase.com
2. **Войти** или зарегистрироваться

   - Region: `Europe West (London)` или ближайший

1. **Открыть:** SQL Editor в Supabase
2. **New Query**
3. **Скопировать** содержимое файла `database/migrations/001-add-ml-tables-safe.sql`

3. **Скопировать** строку подключения
4. **Формат:** `postgresql://postgres:[PASSWORD]@[HOST]:5432/postgres`

## 🆘 Если что-то не работает

### Бот не отвечает
1. Проверить webhook: `getWebhookInfo`
2. Проверить логи Vercel: Dashboard → Deployments → Latest → Logs
3. Проверить переменные окружения
4. Переустановить webhook

### Ошибки БД
1. Проверить DATABASE_URL
2. Проверить, что миграция применена
3. Проверить логи Supabase: Dashboard → Logs

### Mini App не открывается
1. Проверить MINIAPP_URL
2. Проверить, что файл elite.html существует
3. Открыть в браузере: `https://[ваш-url].vercel.app/miniapp/elite.html`
