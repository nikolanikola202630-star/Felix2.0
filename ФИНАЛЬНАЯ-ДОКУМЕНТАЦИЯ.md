## 🎯 Обзор проекта

Felix Academy - это образовательная платформа на базе Telegram, объединяющая:

- 🤖 **Telegram бот** с AI-ассистентом (Groq Llama 3.3 70B)
- 📱 **Mini App** с современным интерфейсом
- 🎓 **Система курсов** с видео-уроками
- 💰 **Платежи** через Telegram Stars
- 👥 **Партнерская программа** (20% комиссия)
- 📊 **Аналитика** и статистика
- 🛡️ **Безопасность** (rate limiting, валидация, CSRF защита)

### Ключевые метрики

```
📊 Статистика проекта:
├─ Курсов: 3 (55 уроков, 70 часов)
├─ Студентов: 3,293
├─ Выручка: 150,000₽
```

## 🚀 Установка и запуск

### Требования

- Node.js 18+
- npm или yarn
- Telegram Bot Token
- Groq API Key
- Supabase проект (опционально)

### Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone https://github.com/your-username/felix-academy.git
cd felix-academy

# 2. Установить зависимости
npm install

# 3. Создать .env файл
cat > .env << EOF
TELEGRAM_BOT_TOKEN=your_bot_token
GROQ_API_KEY=your_groq_key
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_key
NODE_ENV=development
EOF

# 4. Запустить бота локально
node bot.js

# 5. В другом терминале - запустить dev сервер
npm run dev

# 6. Открыть в браузере
# http://localhost:3000/miniapp/index.html
```

### Настройка Telegram бота

```bash
# 1. Создать бота через @BotFather
/newbot
# Следовать инструкциям

# 2. Получить токен
# Сохранить в .env как TELEGRAM_BOT_TOKEN

# 3. Настроить Mini App
/mybots → выбрать бота → Bot Settings → Menu Button
# URL: https://your-domain.vercel.app/miniapp/index.html

# 4. Настроить Telegram Stars (для платежей)
/mybots → выбрать бота → Payments → Telegram Stars
```

### Настройка Supabase

```bash
# 1. Создать проект на supabase.com

# 2. Применить миграции
psql -h your-host -U postgres -d postgres -f database/schema.sql
psql -h your-host -U postgres -d postgres -f database/migrations/002-academy-tables.sql
psql -h your-host -U postgres -d postgres -f database/migrations/003-partner-courses.sql

# 3. Получить URL и ключи
# Project Settings → API
# Сохранить в .env
```

## 📡 API документация

### Базовый URL

```
Production: https://felix2-0.vercel.app/api
Local: http://localhost:3000/api
```

### Аутентификация

Все API требуют либо:
- `userId` в query/body (для проверки прав)
- `X-Telegram-WebApp-Init-Data` header (для Mini App)

### Endpoints

#### Курсы

**GET /api/courses-full**
```javascript
// Получить все курсы
GET /api/courses-full

// Фильтры
GET /api/courses-full?category=psychology
GET /api/courses-full?level=beginner
GET /api/courses-full?id=1

// Ответ
{
  "success": true,
  "count": 3,
  "courses": [...]
}
```

**GET /api/courses/check-access**
```javascript
// Проверить доступ к курсу
GET /api/courses/check-access?user_id=123&course_id=1

// Ответ
{
  "success": true,
  "has_access": true,
  "reason": "purchased",
  "lessons": [...],
  "stats": {
    "total_lessons": 5,
    "accessible_lessons": 5,
    "locked_lessons": 0
  }
}
```

#### Платежи

**POST /api/payments/create-invoice**
```javascript
// Создать счет для оплаты
POST /api/payments/create-invoice
{
  "course_id": 1,
  "user_id": 123456
}

// Ответ
{
  "success": true,
  "invoice_link": "https://t.me/$...",
  "course": {
    "id": 1,
    "title": "...",
    "price_rub": 2990,
    "price_stars": 1495
  }
}
```

**POST /api/payments/webhook**
```javascript
// Webhook от Telegram (автоматический)
// Обрабатывает успешные платежи
// Начисляет комиссии
// Отправляет уведомления
```

**POST /api/payments/refund**
```javascript
// Возврат средств (только админы)
POST /api/payments/refund
{
  "purchase_id": "...",
  "reason": "Причина возврата",
  "admin_id": 8264612178
}
```

#### Админ панель

**POST /api/admin/courses-manage**
```javascript

  "action": "updatePrice",

  "action": "bulkUpdatePrices",

{

  "themeId": 1,
  "lesson": {
    "title": "Новый урок",
    "duration": 45,
    "is_free": false,
    "video_url": "..."
  }

// Статистика

```

#### Партнерка

**POST /api/partner**
```javascript

{
  "action": "getStudents",

// Отправить сообщение студенту
```

### Rate Limiting

```
API endpoints: 100 запросов/минуту
AI endpoints: 10 запросов/минуту
Payment endpoints: 5 запросов/5 минут
Webhook: 200 запросов/минуту
```

При превышении лимита:
```json
{
  "success": false,
  "error": "Too many requests",
  "retryAfter": 60
}
```

## 💰 Платежи

### Telegram Stars

**Настройка:**
1. @BotFather → /mybots → выбрать бота
2. Payments → Telegram Stars
3. Готово! (provider_token не нужен)

**Конвертация:**
```
1 Star ≈ 2 рубля
2990₽ = ~1495 Stars
3990₽ = ~1995 Stars
4990₽ = ~2495 Stars
```

**Флоу оплаты:**
```
1. Пользователь нажимает "Купить"
2. API создает invoice
3. Mini App открывает invoice через tg.openInvoice()
4. Пользователь оплачивает
5. Telegram отправляет webhook
6. API сохраняет покупку
7. API начисляет комиссии
8. Пользователь получает уведомление
9. Курс становится доступен
```

**Комиссии:**
- Реферер: 20% от суммы покупки
- Партнер курса: настраивается индивидуально
- Платформа: остаток

**Возврат средств:**
- Доступен только админам
- Через API `/api/payments/refund`
- Автоматическое уведомление пользователя

## 🚀 Развертывание

### Vercel (рекомендуется)

```bash
# 1. Установить Vercel CLI
npm i -g vercel

# 2. Залогиниться
vercel login

# 3. Связать проект
vercel link

# 4. Добавить переменные окружения
vercel env add TELEGRAM_BOT_TOKEN
vercel env add GROQ_API_KEY
vercel env add SUPABASE_URL
vercel env add SUPABASE_KEY

# 5. Задеплоить
vercel --prod
```

### Настройка домена

```bash
# 1. Добавить домен в Vercel
vercel domains add academy.your-domain.com

# 2. Настроить DNS
# A record: @ → 76.76.21.21
# CNAME: www → cname.vercel-dns.com

# 3. Обновить URL в боте
# @BotFather → Menu Button → новый URL
```


**Отслеживать:**
- Количество покупок
- Выручка
- Конверсия

- Rate limit hits

### Sentry (рекомендуется)

```bash

npm install @sentry/node
```
