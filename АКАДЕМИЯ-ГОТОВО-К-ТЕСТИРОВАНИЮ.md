## 🧪 Как протестировать

### 1. Бот уже запущен!

Бот работает в фоне. Проверь логи:

```powershell
# Логи показывают:
🎓 Академия Bot - Local Polling Mode Started!
📍 Mini App: https://felix2-0.vercel.app/miniapp/index.html
🤖 Bot is running and waiting for messages...
```

### 2. Открой бота в Telegram

Найди своего бота и отправь команды:

```
/start
```
**Ожидается:** Приветствие Академии с кнопкой "Открыть Академию"

```
/start ref_partner123
```
**Ожидается:** Приветствие с упоминанием реферальной ссылки

```
/profile
```
**Ожидается:** Профиль с реферальной ссылкой и бонусами

```
/partner
```
**Ожидается:** Партнерский кабинет со статистикой

```
/help
```
**Ожидается:** Список команд Академии

### 3. Открой Mini App

Нажми кнопку "🎓 Открыть Академию" в боте

**Ожидается:**
- Приветствие с твоим именем
- Раздел "Бесплатные уроки"
- Раздел "Популярные курсы"
- Нижняя навигация
