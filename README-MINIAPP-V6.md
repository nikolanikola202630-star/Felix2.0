## 🚀 Как использовать?

### 1. Открыть Mini App
```
1. Откройте бота в Telegram
2. Отправьте команду /start
3. Нажмите кнопку "🚀 Открыть Mini App"
```

### 2. Навигация
- Используйте вкладки вверху для переключения разделов
- Нажимайте на карточки для детальной информации
- Свайпайте для быстрой навигации

### 3. Курсы
```
1. Перейдите в "🎓 Академия"
2. Выберите курс
3. Нажмите "🚀 Начать курс"
4. Проходите уроки
5. Получайте XP
```

### 4. Персонализация
```
1. Перейдите в "⚙️ Настройки"
2. Выберите аватар или эмодзи
3. Настройте стиль общения
4. Выберите язык и тему
5. Нажмите "💾 Сохранить"
```

### 5. Голосовое управление
```
1. Нажмите кнопку 🎤 внизу справа
2. Произнесите команду
3. Например: "Открыть академию"
```

## 🔧 Технические детали

### Технологии
- HTML5
- CSS3 (Variables, Animations, Glassmorphism)
- JavaScript (ES6+)
- Telegram WebApp SDK
- Fetch API

### API Endpoints
```
/api/miniapp    - Профиль и настройки
/api/courses    - Курсы и прогресс
/api/community  - Сообщество и события
/api/learning   - Обучение и достижения
/api/voice      - Голосовое управление
```

### Локальные данные
```
miniapp/courses-full.json   - Полные данные курсов
miniapp/partners-full.json  - Полные данные партнеров
miniapp/community.json      - Данные сообщества
```

### Fallback
- Автоматический fallback на локальные данные
- Работа без интернета (частично)
- Graceful degradation

## 🐛 Troubleshooting

### Проблема: Не загружается
**Решение**: 
1. Проверьте интернет соединение
2. Обновите страницу
3. Очистите кэш браузера

### Проблема: Курсы не отображаются
**Решение**:
1. Проверьте консоль на ошибки
2. Убедитесь, что API доступен
3. Проверьте наличие courses-full.json

### Проблема: Анимации тормозят
**Решение**:
1. Закройте другие приложения
2. Обновите Telegram
3. Перезагрузите устройство

### Проблема: Голос не работает
**Решение**:
1. Разрешите доступ к микрофону
2. Проверьте настройки браузера
3. Используйте HTTPS
