## 📋 Структура курса:

### Основная информация:
```javascript
{
  id: 1,                          // Уникальный ID
  slug: "psychology",             // URL-friendly название
  title: "Название курса",        // Полное название
  category: "psychology",         // Категория
  description: "Описание...",     // Полное описание
  price: 2990,                    // Цена в рублях
  rating: 4.8,                    // Рейтинг (0-5)
  students: 1247,                 // Количество студентов
  duration_hours: 15,             // Длительность в часах
  level: "beginner",              // Уровень сложности
  image: "https://...",           // URL изображения
  instructor: {                   // Преподаватель
    name: "Имя",
    title: "Должность",
    avatar: "👨‍🎓"
  }
}
```

### Темы и уроки:
```javascript
themes: [
  {
    id: 1,
    title: "Название темы",
    file: "tema1.pdf",            // Файл с материалами
    lessons: [
      {
        id: 1,
        title: "Название урока",
        duration: 45,             // Минуты
        is_free: true,            // Бесплатный урок
        video_url: "https://...", // URL видео
        description: "..."        // Описание урока
      }
    ]
  }
]
```

## 🔧 API Endpoints:

### Получить все курсы:
```
GET /api/admin-courses?action=getCourses&userId=YOUR_ID
```

### Сохранить курсы:
```
POST /api/admin-courses
{
  "action": "saveCourses",
  "userId": YOUR_ID,
  "courses": [...]
}
```

### Добавить курс:
```
POST /api/admin-courses
{
  "action": "addCourse",
  "userId": YOUR_ID,
  "course": {...}
}
```

### Обновить курс:
```
POST /api/admin-courses
{
  "action": "updateCourse",
  "userId": YOUR_ID,
  "courseId": 1,
  "updates": {...}
}
```

### Удалить курс:
```
POST /api/admin-courses
{
  "action": "deleteCourse",
  "userId": YOUR_ID,
  "courseId": 1
}
```

### Получить статистику:
```
GET /api/admin-courses?action=getStats&userId=YOUR_ID
```

### Список бэкапов:
```
GET /api/admin-courses?action=listBackups&userId=YOUR_ID
```

### Восстановить бэкап:
```
POST /api/admin-courses
{
  "action": "restoreBackup",
  "userId": YOUR_ID,
  "backupFile": "courses-backup-1234567890.json"
}
```

## 🔐 Безопасность:

### Проверка прав:
- Только пользователи из списка `ADMINS` могут управлять курсами
- Проверка `userId` на каждый запрос
- Автоматическое создание бэкапов при сохранении

### Список админов:
Редактируйте файл `api/admin-courses.js`:
```javascript
const ADMINS = [
  8264612178,  // Ваш Telegram ID
  123456789,   // Другой админ
];
```

## 💾 Резервное копирование:

### Автоматические бэкапы:
- Создаются при каждом сохранении
- Хранятся в папке `data/`
- Формат: `courses-backup-TIMESTAMP.json`

### Восстановление:
1. Откройте админ-панель
2. Нажмите "Список бэкапов"
3. Выберите нужный бэкап
4. Нажмите "Восстановить"

## 📝 Пошаговая инструкция:

### Добавление нового курса:

1. **Откройте админ-панель**
   - Перейдите на `admin-courses.html`

2. **Нажмите "Добавить курс"**
   - Откроется модальное окно

3. **Заполните основную информацию:**
   - Название курса
   - Slug (URL)
   - Описание
   - Категория
   - Уровень сложности
   - Цена
   - Длительность

4. **Добавьте информацию о преподавателе:**
   - Имя
   - Должность
   - Эмодзи (опционально)

5. **Добавьте темы:**
   - Нажмите "+ Добавить тему"
   - Введите название темы
   - Укажите файл PDF (опционально)

6. **Добавьте уроки к каждой теме:**
   - Нажмите "+ Добавить урок"
   - Название урока
   - Длительность (минуты)
   - URL видео
   - Отметьте "Бесплатный" если нужно

7. **Сохраните курс:**
   - Нажмите "Сохранить курс"
   - Курс появится в списке

## 🐛 Решение проблем:

### Проблема: Не открывается админ-панель
**Решение:**
- Проверьте URL
- Убедитесь что вы в списке админов
- Проверьте консоль браузера (F12)

### Проблема: Не сохраняются изменения
**Решение:**
- Проверьте права доступа
- Убедитесь что все обязательные поля заполнены
- Проверьте логи сервера

### Проблема: Не загружаются курсы
**Решение:**
- Проверьте файл `data/courses-structure.json`
- Убедитесь что JSON валидный
- Проверьте API endpoint
