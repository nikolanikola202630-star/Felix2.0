### Courses Integration
```javascript
// Загрузка курсов с прогрессом
loadCourses() → Courses API → courses-full.json (fallback)

// Детали курса
openCourseDetail(id) → Уроки + Прогресс + Типы

// Начать курс
startCourseNew(id) → API → +10 XP

// Завершить урок
completeLessonNew(id) → API → +20 XP → Обновление
```
