## 💬 Система чатов:

### Как работает:

1. **Студент начинает обучение**
   - Автоматически создается связь с партнером
   - Партнер видит студента в списке

2. **Партнер открывает чат**
   - Нажимает "Написать" у студента
   - Открывается окно чата
   - Может отправлять сообщения

3. **Студент получает уведомление**
   - Через Telegram бота
   - С кнопкой "Открыть чат"
   - Может ответить партнеру

4. **Общение в реальном времени**
   - Сообщения сохраняются
   - История доступна всегда
   - Уведомления о новых сообщениях

## 📁 Структура базы данных:

### Таблица partners:
```sql
CREATE TABLE partners (
  id BIGSERIAL PRIMARY KEY,
  user_id BIGINT NOT NULL UNIQUE,
  username TEXT,
  first_name TEXT,
  status TEXT DEFAULT 'active',
  commission_rate DECIMAL(5,2) DEFAULT 20.00,
  total_earnings DECIMAL(10,2) DEFAULT 0,
  total_students INTEGER DEFAULT 0,
  created_at TIMESTAMP DEFAULT NOW()
);
```

### Таблица course_partners:
```sql
CREATE TABLE course_partners (
  id BIGSERIAL PRIMARY KEY,
  course_id INTEGER NOT NULL,
  partner_id BIGINT NOT NULL,
  role TEXT DEFAULT 'owner',
  can_edit BOOLEAN DEFAULT true,
  can_view_students BOOLEAN DEFAULT true,
  can_chat BOOLEAN DEFAULT true,
  UNIQUE(course_id, partner_id)
);
```

### Таблица course_students:
```sql
CREATE TABLE course_students (
  id BIGSERIAL PRIMARY KEY,
  course_id INTEGER NOT NULL,
  user_id BIGINT NOT NULL,
  partner_id BIGINT,
  status TEXT DEFAULT 'active',
  progress INTEGER DEFAULT 0,
  started_at TIMESTAMP DEFAULT NOW(),
  last_activity TIMESTAMP DEFAULT NOW(),
  UNIQUE(course_id, user_id)
);
```

### Таблица partner_chats:
```sql
CREATE TABLE partner_chats (
  id BIGSERIAL PRIMARY KEY,
  partner_id BIGINT NOT NULL,
  student_id BIGINT NOT NULL,
  course_id INTEGER,
  status TEXT DEFAULT 'open',
  last_message_at TIMESTAMP DEFAULT NOW(),
  UNIQUE(partner_id, student_id, course_id)
);
```

### Таблица partner_messages:
```sql
CREATE TABLE partner_messages (
  id BIGSERIAL PRIMARY KEY,
  chat_id BIGINT NOT NULL,
  sender_id BIGINT NOT NULL,
  sender_type TEXT NOT NULL,
  message TEXT NOT NULL,
  is_read BOOLEAN DEFAULT false,
  created_at TIMESTAMP DEFAULT NOW()
);
```

## 🔧 API Endpoints:

### Получить профиль партнера:
```
GET /api/partner?action=getProfile&userId=YOUR_ID
```

### Получить курсы партнера:
```
GET /api/partner?action=getMyCourses&userId=YOUR_ID
```

### Получить студентов:
```
GET /api/partner?action=getMyStudents&userId=YOUR_ID
```

### Получить чаты:
```
GET /api/partner?action=getMyChats&userId=YOUR_ID
```

### Получить сообщения чата:
```
GET /api/partner?action=getChatMessages&userId=YOUR_ID&chatId=CHAT_ID
```

### Отправить сообщение:
```
POST /api/partner
{
  "action": "sendMessage",
  "userId": YOUR_ID,
  "chatId": CHAT_ID,
  "message": "Текст сообщения"
}
```

### Создать чат:
```
POST /api/partner
{
  "action": "createChat",
  "userId": YOUR_ID,
  "studentId": STUDENT_ID,
  "courseId": COURSE_ID
}
```

### Получить прогресс студента:
```
GET /api/partner?action=getStudentProgress&userId=YOUR_ID&studentId=STUDENT_ID&courseId=COURSE_ID
```

### Получить статистику:
```
GET /api/partner?action=getStats&userId=YOUR_ID
```

## 💰 Система комиссий:

### Как работает:

1. **Студент покупает курс**
   - Цена курса: 2990₽
   - Комиссия партнера: 20% = 598₽
   - Комиссия платформы: 80% = 2392₽

2. **Начисление комиссии**
   - Автоматически после оплаты
   - Добавляется к балансу партнера
   - Доступно для вывода

3. **Вывод средств**
   - Минимальная сумма: 1000₽
   - На карту или крипту
   - Обработка: 1-3 дня
