## 🎯 Ключевые изменения

### 1. Цветовая схема

**Было:**
```css
--bg-primary: #1A1A1A;
--primary: #D4AF37; /* Золото */
--accent: #10B981; /* Изумруд */
```

**Стало:**
```css
--bg-primary: #0A0A0A; /* Чернее */
--accent: #FFFFFF; /* Белый вместо золота */
--text-primary: #FFFFFF;
```

**Результат:** Строже, минималистичнее, контрастнее

### 2. Типографика

**Было:**
```css
--font-display: 'Playfair Display', serif;
--font-accent: 'Cormorant Garamond', serif;
--font-sans: 'Montserrat', sans-serif;
--text-h1: 36px;
```

**Стало:**
```css
--font-sans: -apple-system, 'SF Pro Display', 'Roboto', sans-serif;
--text-hero: 48px; /* +33% */
--text-h1: 36px;
--font-black: 900; /* Жирнее */
```

**Результат:** Системные шрифты, крупнее, жирнее

### 3. Компоненты

**Было:**
```css
.card {
  background: linear-gradient(...);
  border: 2px solid #D4AF37;
  box-shadow: 0 10px 30px rgba(212, 175, 55, 0.2);
}
```

**Стало:**
```css
.card {
  background: #1A1A1A;
  border: 1px solid #333333;
  /* Без теней и градиентов */
}
```

**Результат:** Плоский дизайн, минимум эффектов

### 4. Кнопки

**Было:**
```css
.btn-primary {
  background: linear-gradient(135deg, #D4AF37, #B8941F);
  box-shadow: 0 4px 15px rgba(212, 175, 55, 0.3);
}
```

**Стало:**
```css
.btn-primary {
  background: #FFFFFF;
  color: #0A0A0A;
  /* Без градиентов и теней */
}
```

**Результат:** Четкие, контрастные, без излишеств

### 5. Анимации

**Было:**
- Сложные spring анимации
- Stagger эффекты
- Backdrop blur
- Множество transition

**Стало:**
- Простые fade-in
- Базовые hover эффекты
- Минимум анимаций
- Быстрые transition (150ms)

**Результат:** Быстрее, проще, четче
