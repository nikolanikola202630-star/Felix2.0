# 👀 Как Увидеть Обновления MiniApp

## 🔥 Быстрое Решение

### 1. Жесткая перезагрузка браузера
```
Windows: Ctrl + Shift + R
Mac: Cmd + Shift + R
```

### 2. Очистить кэш браузера
```
Chrome/Edge:
1. F12 (открыть DevTools)
2. Правый клик на кнопке обновления
3. "Очистить кэш и жесткая перезагрузка"
```

### 3. Открыть в режиме инкогнито
```
Windows: Ctrl + Shift + N
Mac: Cmd + Shift + N
```

## 📱 Для Telegram WebApp

### Если открываешь через Telegram:

1. **Закрой MiniApp полностью**
2. **Перезапусти Telegram**
3. **Открой MiniApp заново**

### Альтернатива:
```
Telegram → Settings → Advanced → Clear Cache → Clear
```

## 🔍 Проверка в DevTools

### 1. Открой консоль (F12)
### 2. Вкладка "Network"
### 3. Обнови страницу
### 4. Найди footer.js

**Должно быть:**
- Status: 200 (зеленый)
- Size: ~1.2 KB
- Time: < 100ms

**Если видишь:**
- Status: 304 (серый) - это кэш, нужна жесткая перезагрузка
- Status: 404 (красный) - файл не найден, проблема с деплоем

## 🐛 Если Не Работает

### 1. Проверь версию файла:
```javascript
// В консоли браузера (F12)
console.log(document.querySelector('script[src*="footer"]')?.src);
// Должно быть: .../footer.js?v=202603031830
```

### 2. Проверь ошибки:
```javascript
// В консоли браузера
// Не должно быть ошибок типа:
// "Failed to load resource: footer.js"
// "Uncaught SyntaxError"
```

### 3. Проверь футер в DOM:
```javascript
// В консоли браузера
console.log(document.querySelector('.app-footer'));
// Должен вернуть: <footer class="app-footer">...</footer>
```
