## 🎯 Общая концепция {#концепция}

### Что это за проект?
Felix Bot - это **умный Telegram-бот с AI-ассистентом**, который:
- Общается с пользователями через Groq AI (Llama 3.3 70B)
- Запоминает всю историю диалогов в PostgreSQL
- Имеет красивый Mini App интерфейс
- Предлагает систему обучения с уровнями и достижениями
- Поддерживает курсы, партнеров, персонализацию

### Целевая аудитория
- Пользователи, которым нужен AI-помощник в Telegram
- Студенты, изучающие курсы
- Группы, требующие модерации
- Партнеры, предоставляющие услуги

### Уникальность
1. **Персонализация**: Адаптируется под стиль общения (formal/casual)
2. **Система прогресса**: Уровни, XP, достижения, стрики
3. **Mini App**: Полноценный веб-интерфейс с анимациями
4. **База данных**: Полная история и контекст для AI
5. **Модерация**: Автоматическая модерация групп

### Компоненты

#### 1. Backend (Vercel Serverless)
**13 API endpoints:**
- `webhook.js` - Основной обработчик сообщений
- `admin.js` - Админ-панель API
- `learning.js` - Система обучения
- `history.js` - История сообщений
- `search.js` - Поиск по сообщениям
- `stats.js` - Статистика
- `settings.js` - Настройки пользователя
- `voice.js` - Голосовые сообщения
- `export.js` - Экспорт данных
- `analytics.js` - Аналитика
- `courses.js` - Курсы
- `community.js` - Сообщество
- `miniapp.js` - Mini App API


#### 2. Database (PostgreSQL/Supabase)
**9 таблиц:**
- `users` - Профили пользователей
- `messages` - История сообщений (с full-text search)
- `tags` - Теги для категоризации
- `message_tags` - Связь сообщений и тегов
- `user_settings` - Настройки пользователей
- `voice_messages` - Метаданные голосовых
- `image_messages` - Метаданные изображений
- `document_messages` - Метаданные документов
- `export_history` - История экспортов

**15+ индексов** для оптимизации:
- B-tree индексы для быстрого поиска
- GIN индексы для full-text search (русский язык)
- Trigram индексы для нечеткого поиска
- Composite индексы для сложных запросов

#### 3. AI Module (lib/ai.js)
**Провайдер:** Groq (Llama 3.3 70B)

**Функции:**
- `getChatResponse()` - Ответы с контекстом истории
- `createSummary()` - Саммари диалогов (brief/medium/detailed)
- `analyzeText()` - Анализ текста (sentiment, keywords, topics)
- `generateContent()` - Генерация контента (article/email/social/code)
- `organizeText()` - Структурирование текста

**Особенности:**
- Учитывает последние 10 сообщений для контекста
- Настраиваемая температура (0.0-2.0)
- Поддержка русского и английского
- Метрики: tokens, latency

#### 4. Mini App (miniapp/)
**2 интерфейса:**
- `index.html` (2926 строк) - Основное приложение
- `admin.html` (1033 строки) - Админ-панель

**6 вкладок в основном приложении:**
1. **Профиль** - Прогресс, уровень, XP, стрики, достижения
2. **Обучение** - Ежедневные задания, активные курсы
3. **Аналитика** - Графики активности, топ команды
4. **Рейтинг** - Лидерборд пользователей
5. **Академия** - Курсы с прогрессом
6. **Настройки** - Персонализация (аватар, стиль, тема)

- ❌ Нет обработки SIGTERM
- ❌ Соединения с БД могут не закрыться
- ❌ Потеря данных при рестарте

**Решение:**
```javascript
// Graceful shutdown
process.on('SIGTERM', async () => {
  console.log('SIGTERM received, closing connections...');
  await pool.end();
  process.exit(0);
});
```

**Приоритет:** 🟡 Средний

  constructor() {
    this.limits = new Map();
  }

  async checkLimit(userId) {
    const today = new Date().toDateString();
    const key = `${userId}:${today}`;
    const count = this.limits.get(key) || 0;
    
    if (count >= 50) {
      return { allowed: false, remaining: 0 };
    }
    
    this.limits.set(key, count + 1);
    return { allowed: true, remaining: 50 - count - 1 };
  }
