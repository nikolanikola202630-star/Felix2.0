## 🎯 О проекте

EGOIST ACADEMY — образовательная платформа в формате Telegram Mini App, объединяющая:

- 📚 Обучающие курсы (PDF + YouTube видео)
- 👥 Тематические сообщества
- 🤖 AI-голосовой ассистент
- 💼 Партнёрская программа
- 📊 Отслеживание прогресса

---

## ✨ Особенности

### Дизайн
- **Минимализм:** Ничего лишнего, фокус на контенте
- **Тёмная тема:** #0A0A0A фон, белый текст
- **Крупные заголовки:** 48px hero, жирные начертания
- **Системные шрифты:** SF Pro Display, Roboto

### Функционал
- ✅ 5 категорий курсов (расширяемо до 16+)
- ✅ YouTube видео + PDF материалы
- ✅ Telegram-сообщества для каждого курса
- ✅ AI-ассистент с голосовым вводом
- ✅ Партнёрская программа с реферальной системой
- ✅ Админ панель для управления

## 🚀 Быстрый старт

### Требования
- Node.js 18+
- Telegram Bot Token
- Vercel аккаунт

### Установка

1. **Клонировать репозиторий**
```bash
git clone <repo-url>
cd <repo-name>
```

2. **Настроить переменные окружения**
```env
MINIAPP_URL=https://your-domain.vercel.app/miniapp/egoist.html
TELEGRAM_BOT_TOKEN=your_bot_token
DATABASE_URL=your_database_url
GROQ_API_KEY=your_groq_key
```

3. **Создать Telegram-группы**
- EGOIST ACADEMY KINGS
- EGOIST BRAIN
- EGOIST IT
- EGOIST TRADING
- EGOIST SUCCESS

4. **Обновить ссылки на группы**

В файле `miniapp/js/egoist-app.js`:
```javascript
renderCommunityCard('ACADEMY KINGS', '👑', 'https://t.me/+YOUR_LINK')
// ... остальные группы
```

5. **Деплой**
```bash
git add .
git commit -m "feat: setup EGOIST ACADEMY"
git push
```

Vercel автоматически задеплоит проект.

## 🎨 Дизайн-система

### Цвета

```css
--bg-primary: #0A0A0A;      /* Фон */
--bg-secondary: #1A1A1A;    /* Карточки */
--text-primary: #FFFFFF;    /* Основной текст */
--text-secondary: #B0B0B0;  /* Вторичный текст */
--border: #333333;          /* Границы */
```

### Типографика

```css
--font-sans: SF Pro Display, Roboto;
--text-hero: 48px;          /* Главные заголовки */
--text-h1: 36px;            /* H1 */
--text-h2: 28px;            /* H2 */
--font-bold: 700;           /* Жирный */
--font-black: 900;          /* Очень жирный */
```

### Компоненты

- **Кнопки:** Белые (primary), серые (secondary), прозрачные (ghost)
- **Карточки:** Минималистичные, с тонкими границами
- **Инпуты:** Темные, с фокусом на белой границе
- **Бейджи:** Маленькие, uppercase, с иконками

## 🔧 API

### Endpoints

```
GET  /api/courses-full      # Все курсы
GET  /api/courses/:id       # Конкретный курс
GET  /api/my-courses        # Мои курсы
POST /api/partner/apply     # Заявка на партнёрство
GET  /api/community         # Сообщества
POST /api/voice-assistant   # AI-ассистент
```

### Структура курса

```json
}
```
