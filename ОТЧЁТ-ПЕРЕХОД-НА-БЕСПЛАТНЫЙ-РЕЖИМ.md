## 🔄 Как вернуть платные функции

### Быстрый возврат (5 минут)

1. **Раскомментировать API код**
   ```bash
   # Найти все /* PAYMENT: и раскомментировать
   grep -r "PAYMENT:" api/
   ```

2. **Обновить БД**
   ```sql
   UPDATE courses SET is_free = false WHERE price > 0;
   ```

3. **Обновить фронтенд**
   - Вернуть отображение цен
   - Вернуть кнопки "Купить"

4. **Деплой**
   ```bash
   git commit -m "feat: возврат платных функций"
   git push
   ```

### Подробная инструкция

См. [РЕЖИМ-БЕСПЛАТНОГО-ДОСТУПА.md](РЕЖИМ-БЕСПЛАТНОГО-ДОСТУПА.md) раздел "Как вернуть платные функции"
