## 🤖 Функции бота

### /start
```
Приветствие с описанием
Кнопка Mini App
Кнопки "Начать диалог" и "Статистика"
```

### Текстовые сообщения
```
Сохранение в БД
Загрузка контекста (10 сообщений)
AI ответ с контекстом
Сохранение ответа
```

### Голосовые сообщения
```
Скачивание файла
Распознавание через Whisper
Сохранение транскрипции
AI ответ на распознанный текст
```

### /organize
```
Получение текста
Структурирование через AI
Сохранение в БД
Отправка результата
```

### /clear
```
Удаление всех сообщений пользователя
Подтверждение
```

## 🐛 Отладка

### Если Mini App не загружается
1. Открыть консоль браузера (F12)
2. Проверить ошибки JavaScript
3. Проверить сетевые запросы
4. Проверить что API endpoints доступны

### Если данные не отображаются
1. Проверить что таблицы созданы в Supabase
2. Проверить DATABASE_URL в Vercel
3. Проверить логи Vercel
4. Проверить что userId передается

### Если кнопки не работают
1. Проверить консоль на ошибки
2. Проверить что Telegram WebApp инициализирован
3. Проверить что обработчики событий добавлены
