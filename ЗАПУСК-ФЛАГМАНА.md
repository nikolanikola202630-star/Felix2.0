## 🐛 Troubleshooting

### Проблема: Бот не отвечает

**Решение:**
```bash
# Проверить webhook
curl "https://api.telegram.org/bot<TOKEN>/getWebhookInfo"

# Удалить webhook (для локального запуска)
curl -X POST "https://api.telegram.org/bot<TOKEN>/deleteWebhook"

# Перезапустить бота
node bot-academy-local.js
```

### Проблема: Mini App не открывается

**Решение:**
1. Проверить MINIAPP_URL в боте
2. Проверить деплой на Vercel
3. Открыть URL в браузере
4. Проверить консоль браузера

### Проблема: AI не отвечает

**Решение:**
1. Проверить GROQ_API_KEY
2. Проверить лимиты API
3. Проверить консоль браузера
4. Попробовать другой вопрос

### Проблема: БД не работает

**Решение:**
```bash
# Проверить подключение
psql $DATABASE_URL -c "SELECT 1"

const cache = new Map();

  cache.set('courses', courses);

```

**Lazy loading:**
```javascript
// Загружать изображения по мере прокрутки
const images = document.querySelectorAll('img[data-src]');
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      const img = entry.target;
      img.src = img.dataset.src;
      observer.unobserve(img);
    }
  });
});

images.forEach(img => observer.observe(img));
```

// sw.js
self.addEventListener('install', (event) => {
  event.waitUntil(
    caches.open('felix-v1').then((cache) => {
      return cache.addAll([
        '/miniapp/flagship.html',
        '/miniapp/js/flagship.js',
        '/miniapp/catalog.html'
      ]);
    })
  );
});

self.addEventListener('fetch', (event) => {
  event.respondWith(
    caches.match(event.request).then((response) => {
      return response || fetch(event.request);
    })
  );
});
