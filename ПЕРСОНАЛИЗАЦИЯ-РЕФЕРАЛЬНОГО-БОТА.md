## 📋 Доступные Условия Доступа

### 1. Подписка на Telegram канал ✅
```json
{
  "type": "telegram_subscription",
  "channel": "@your_channel",
  "required": true
}
```

**Как работает:**
1. Пользователь переходит по реф-ссылке
2. Бот просит подписаться на канал
3. Пользователь подписывается
4. Нажимает "Я подписался"
5. Бот проверяет подписку
6. Доступ открыт!

### 2. Квиз ✅
```json
{
  "type": "quiz",
  "min_score": 80,
  "questions": [...]
}
```

**Как работает:**
1. Пользователь нажимает "Пройти квиз"
2. Бот задает вопросы
3. Пользователь отвечает
4. Если набрал >= 80% - доступ открыт

### 3. Анкета ✅
```json
{
  "type": "form",
  "fields": [
    {"name": "name", "type": "text", "required": true},
    {"name": "email", "type": "email", "required": true}
  ]
}
```

**Как работает:**
1. Пользователь нажимает "Заполнить анкету"
2. Бот запрашивает данные
3. Пользователь заполняет
4. Доступ открыт!

## 💬 Приветственное Сообщение

Кастомизируй текст приветствия:

```
welcome_message: "Привет! 👋 Добро пожаловать в мою академию! 
Здесь ты научишься всему, что нужно для успеха!"
```

**Переменные:**
- `{name}` - имя пользователя
- `{username}` - username пользователя

## 📨 Автоматические Сообщения

Настрой авто-сообщения:

```json

  {
    "trigger": "after_subscription",
    "delay_minutes": 5,
    "message": "Спасибо за подписку! Вот бонус..."
  },
  {
    "trigger": "after_quiz",
    "message": "Отлично! Ты прошел квиз!"
  },
  {
    "trigger": "after_form",
    "message": "Анкета получена! Скоро свяжемся."
  }
]
```

## 🔌 API Endpoints

### Получить настройки
```
GET /api/partner-referral-settings?partner_user_id=123
```

### Обновить настройки
```
POST /api/partner-referral-settings?partner_user_id=123
{
  "welcome_message": "...",
  "bot_name": "...",
  "access_conditions": [...]
}
```

### Управление квизом
```
GET /api/partner-referral-settings/quiz?partner_user_id=123
POST /api/partner-referral-settings/quiz?partner_user_id=123
DELETE /api/partner-referral-settings/quiz/1?partner_user_id=123
```

### Управление анкетой
```
GET /api/partner-referral-settings/form?partner_user_id=123
POST /api/partner-referral-settings/form?partner_user_id=123
DELETE /api/partner-referral-settings/form/1?partner_user_id=123
```

### Лог доступа
```

```

## 💾 База Данных

### Новые таблицы:
- `partner_referral_settings` - Настройки бота
- `partner_access_log` - Лог выполнения условий
- `partner_quiz_questions` - Вопросы квиза
- `partner_quiz_results` - Результаты квиза
- `partner_form_fields` - Поля анкеты
- `partner_form_responses` - Ответы на анкету
