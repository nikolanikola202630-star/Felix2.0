## 🚀 Деплой

### Автоматический деплой
Проект настроен на автоматический деплой через GitHub Actions:
1. Push в main → автоматический деплой на Vercel
2. Webhook автоматически обновится

### Ручной деплой (если нужно)
```bash
# 1. Деплой на Vercel
vercel --prod

# 2. Установка webhook
node scripts/set-webhook.js

# 3. Проверка
curl https://your-domain.vercel.app/api/health/database
```
