## 🚀 Процесс деплоя

### Шаг 1: Git Push ✅
```bash
git add .
git commit -m "feat: v10 - community, personalization, referral bot v2, EGOIST branding"
git push origin main
```

### Шаг 2: Ждать Vercel (2-3 минуты) ⏳
Vercel автоматически:
- Обнаружит изменения
- Запустит сборку
- Задеплоит на production
- Обновит все endpoints

### Шаг 3: Применить миграцию БД ⏳
1. Открыть https://supabase.com/dashboard
2. SQL Editor
3. Скопировать `database/migrations/006-partner-referral-customization-simple.sql`
4. Вставить и нажать RUN

