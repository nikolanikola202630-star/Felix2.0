## 🚀 Инструкция по деплою

### Шаг 1: Коммит изменений (2 мин)

```bash
git add -A
git commit -m "feat: v9.0 production ready - full rewrite and optimization"
git push origin main
```

### Шаг 2: База данных (10 мин)

1. Создать проект в Supabase: https://supabase.com
2. Открыть SQL Editor
3. Скопировать и выполнить `database/migrations/001-add-ml-tables-safe.sql`
4. Проверить создание 22 таблиц
5. Скопировать DATABASE_URL из Settings → Database

### Шаг 3: API ключи (5 мин)

1. **Telegram Bot Token:**
   - Открыть @BotFather
   - `/newbot` → следовать инструкциям
   - Скопировать токен

2. **Groq API Key:**
   - Открыть https://console.groq.com
   - API Keys → Create API Key
   - Скопировать ключ

3. **Admin ID:**
   - Открыть @userinfobot
   - Отправить сообщение
   - Скопировать ID

### Шаг 4: Vercel (10 мин)

1. Открыть https://vercel.com
2. New Project → Import Git Repository
3. Выбрать репозиторий felix-bot
4. Configure Project (оставить defaults)
5. Добавить Environment Variables:
   ```
   TELEGRAM_BOT_TOKEN = your_bot_token
   GROQ_API_KEY = your_groq_key
   DATABASE_URL = your_database_url
   ADMIN_ID = your_telegram_id
   MINIAPP_URL = https://your-project.vercel.app/miniapp/elite.html
   NODE_ENV = production
   ```
6. Deploy
7. Скопировать URL проекта
8. Обновить MINIAPP_URL на реальный URL
9. Redeploy

### Шаг 5: Webhook (5 мин)

```bash
curl -X POST "https://api.telegram.org/bot<YOUR_BOT_TOKEN>/setWebhook" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://your-project.vercel.app/api/webhook"}'
```

### Шаг 6: Тестирование (5 мин)

```bash
# Health check
curl https://your-project.vercel.app/api/webhook

# Telegram
# /start, /help, /profile, /stats
# Обычное сообщение
# /ask что такое AI?

# Mini App
# Открыть через кнопку
# Проверить все вкладки
```

**Полная инструкция:** [DEPLOYMENT-V9.md](DEPLOYMENT-V9.md)
