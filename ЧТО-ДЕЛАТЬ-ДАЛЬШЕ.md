## 🔧 НАСТРОЙКА ПОСЛЕ ДЕПЛОЯ:

### Шаг 1: Environment Variables (10 минут)

**Откройте:** Vercel Dashboard → Felix2.0 → Settings → Environment Variables

**Проверьте существующие:**
- ✅ TELEGRAM_BOT_TOKEN
- ✅ GROQ_API_KEY
- ✅ DATABASE_URL
- ✅ ADMIN_ID
- ✅ MINIAPP_URL

**Добавьте новые в v7.1:**

#### Sentry (мониторинг)
```env
SENTRY_DSN=your_sentry_dsn
```

**Как получить:**
1. Создать аккаунт: https://sentry.io
2. Создать проект: Felix Bot
3. Скопировать DSN
4. Добавить в Vercel

#### AI Rate Limiting
```env
AI_DAILY_LIMIT=50
AI_HOURLY_LIMIT=10
```

#### Vercel KV (Redis)
```env
KV_URL=your_kv_url
KV_REST_API_URL=your_kv_api_url
KV_REST_API_TOKEN=your_kv_token
KV_REST_API_READ_ONLY_TOKEN=your_kv_readonly_token
```

**Как получить:**
1. Vercel Dashboard → Storage
2. Create Database → KV (Redis)
3. Скопировать все 4 переменные
4. Добавить в Environment Variables

### Шаг 2: Redeploy с новыми переменными

После добавления переменных:

**Через Dashboard:**
1. Vercel Dashboard → Deployments
2. Последний deployment → ⋯ (три точки)
3. Redeploy

**Через CLI:**
```powershell
vercel --prod --force
```
