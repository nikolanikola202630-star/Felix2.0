## Режимы работы

### Текущий: Polling (локально)
- ✅ Работает на вашем компьютере
- ✅ Не требует Vercel API
- ✅ Мгновенные ответы
- ⚠️ Требует запущенный процесс

### Альтернатива: Webhook (Vercel)
Для использования webhook нужно:
1. Создать рабочий `api/webhook.js`
2. Установить webhook: `node scripts/set-webhook.js`
3. Задеплоить на Vercel

**Но для EGOIST ACADEMY polling достаточно!**
