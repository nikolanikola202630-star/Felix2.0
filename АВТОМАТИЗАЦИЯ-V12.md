## 🔄 Auto Sync V12

### Функции

- **Синхронизация пользователей** - каждые 5 минут
- **Синхронизация курсов** - каждые 5 минут
- **Синхронизация статистики** - каждые 5 минут
- **Очистка кэша** - удаление файлов старше 1 часа
- **Проверка версий файлов** - контроль актуальности

### Использование

```javascript
const AutoSyncV12 = require('./lib/automation/auto-sync-v12');

const sync = new AutoSyncV12();
await sync.start();

// Ручная синхронизация
await sync.syncAll();
```

### API

```bash
# Получить статус синхронизации
GET /api/automation-v12?action=status&user_id=ADMIN_ID

# Запустить синхронизацию
POST /api/automation-v12
{
  "action": "sync",
  "user_id": ADMIN_ID
}
```

## 🏥 Health Monitor V12

### Проверки

- **Эндпоинты** - доступность всех API
- **База данных** - подключение и скорость
- **Память** - использование RAM
- **Диск** - свободное место

### Интервалы

- Проверка здоровья: каждые 2 минуты
- Сохранение результатов: в БД
- Алерты: при критических проблемах

### Статусы

- `healthy` - все работает нормально
- `degraded` - есть проблемы, но система работает
- `critical` - критические проблемы
- `down` - система недоступна

### Использование

```javascript
const HealthMonitorV12 = require('./lib/automation/health-monitor-v12');

const monitor = new HealthMonitorV12();
await monitor.start();

// Получить статус
const status = await monitor.getStatus();
```

### API

```bash
# Получить здоровье системы
GET /api/automation-v12?action=health&user_id=ADMIN_ID&period=24h

# Периоды: 24h, 7d, 30d
```

## 🚀 Auto Deploy V12

### Функции

- **Отслеживание изменений** - проверка Git каждую минуту
- **Автоматический деплой** - при накоплении изменений
- **Оптимизация** - перед деплоем
- **Тесты** - быстрые тесты перед деплоем
- **Уведомления** - о результатах деплоя

### Интервалы

- Минимальный интервал между деплоями: 5 минут
- Проверка изменений: каждую минуту
- Ожидание Vercel: 2 минуты

### Использование

```javascript
const AutoDeployV12 = require('./lib/automation/auto-deploy-v12');

const deploy = new AutoDeployV12();
await deploy.start();

// Ручной деплой
await deploy.manualDeploy();
```

### API

```bash
# Запустить деплой
POST /api/automation-v12
{
  "action": "deploy",
  "user_id": ADMIN_ID
}
```

## ⏰ Запланированные задачи

### Ежедневные (03:00)

- Очистка старых логов (>7 дней)
- Очистка старых health checks (>7 дней)
- Очистка кэша

### Еженедельные (Воскресенье 20:00)

- Генерация отчета за неделю
- Отправка отчета администраторам

### Ежемесячные (1-е число 02:00)

- Оптимизация базы данных
- Полная очистка кэша
- Генерация месячного отчета

## 🚀 Запуск

### Локально

```bash
# Установка зависимостей
npm install

# Запуск автоматизации
node scripts/start-automation-v12.js
```

### Production (Vercel)

Автоматизация запускается через Vercel Cron Jobs:

```json
{
  "crons": [
    {
      "path": "/api/automation-v12?action=sync",
      "schedule": "*/5 * * * *"
    },
    {
      "path": "/api/automation-v12?action=health",
      "schedule": "*/2 * * * *"
    }
  ]
}
```

### Docker

```dockerfile
FROM node:18-alpine

WORKDIR /app
COPY package*.json ./
RUN npm ci --production

COPY . .

CMD ["node", "scripts/start-automation-v12.js"]
```

## 📚 Примеры использования

### Пример 1: Ручная синхронизация

```javascript
const AutoSyncV12 = require('./lib/automation/auto-sync-v12');

async function manualSync() {
  const sync = new AutoSyncV12();
  
  console.log('Starting manual sync...');
  await sync.syncAll();
  console.log('Sync completed!');
}

manualSync();
```

### Пример 2: Проверка здоровья

```javascript
const HealthMonitorV12 = require('./lib/automation/health-monitor-v12');

async function checkHealth() {
  const monitor = new HealthMonitorV12();
  
  const health = await monitor.checkAll();
  console.log('System health:', health.overall);
  
  if (health.overall !== 'healthy') {
    console.warn('System has issues!');
  }
}

checkHealth();
```

### Пример 3: Генерация отчета

```javascript
const MasterAutomationV12 = require('./lib/automation/master-automation-v12');

async function generateReport() {
  const master = new MasterAutomationV12();
  
  const report = await master.generateReport('week');
  console.log('Weekly report:', report);
}

generateReport();
```
