### 1. Добавить Свой ID в Админы

Открой `bot.js` и найди строку:
```javascript
const ADMIN_IDS = [123456789]; // Добавь свой Telegram ID
```

Замени на свой ID:
```javascript
const ADMIN_IDS = [ТВО Й_TELEGRAM_ID];
```

Как узнать свой ID:
1. Напиши боту @userinfobot
2. Скопируй свой ID
3. Вставь в массив

Потом:
```bash
git add bot.js
git commit -m "⚙️ Add admin ID"
git push origin main
```
