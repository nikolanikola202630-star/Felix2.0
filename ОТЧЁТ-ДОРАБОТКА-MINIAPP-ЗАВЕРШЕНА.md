## 🔒 Безопасность

### Реализовано:

1. **Telegram initData валидация**
   - HMAC-SHA256 проверка подлинности
   - Проверка срока действия (24 часа)
   - Защита от replay атак

2. **Централизованная аутентификация**
   - User ID извлекается из валидированных данных
   - Нет передачи user_id в параметрах
   - Опциональная аутентификация для development

3. **Обработка ошибок**
   - Не раскрывает внутренние детали в production
   - Логирование всех ошибок
   - User-friendly сообщения

4. **Rate limiting**
   - Готово к добавлению на уровне API
   - Поддержка на уровне клиента

## 🚀 Деплой

**Команды:**

```bash
# 1. Установить зависимости
npm install

# 2. Настроить переменные окружения
# Добавить в Vercel Dashboard:
# - TELEGRAM_BOT_TOKEN
# - SUPABASE_URL
# - SUPABASE_KEY
# - GROQ_API_KEY

# 3. Задеплоить
vercel --prod

# 4. Установить webhook
curl -X POST "https://api.telegram.org/bot<TOKEN>/setWebhook" \
  -d "url=https://your-app.vercel.app/api/webhook"
```
