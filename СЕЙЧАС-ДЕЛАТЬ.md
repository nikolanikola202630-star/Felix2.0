## 📋 ДЕЙСТВИЕ 2: Подключить к Vercel (5 минут)

1. **Import Project**
   - Vercel Dashboard → Import Project
   - Выбрать GitHub → Felix2.0
   - Deploy

2. **Environment Variables**
   ```env
   # Обязательные
   TELEGRAM_BOT_TOKEN=your_token
   GROQ_API_KEY=your_key
   DATABASE_URL=your_db_url
   ADMIN_ID=your_admin_id
   MINIAPP_URL=your_miniapp_url
   
   # Новые в v7.1
   SENTRY_DSN=your_sentry_dsn
   AI_DAILY_LIMIT=50
   AI_HOURLY_LIMIT=10
   KV_URL=your_kv_url
   KV_REST_API_URL=your_kv_api_url
   KV_REST_API_TOKEN=your_kv_token
   KV_REST_API_READ_ONLY_TOKEN=your_kv_readonly_token
   ```

3. **Проверить деплой**
   - Дождаться завершения
   - Проверить логи
   - ✅ Готово!

## 🔧 ДЕЙСТВИЕ 3: Настроить сервисы (10 минут)

### Vercel KV (Redis)
1. Vercel Dashboard → Storage → Create Database
2. Выбрать KV (Redis)
3. Скопировать переменные окружения
4. Добавить в Vercel Environment Variables

### Sentry (Мониторинг)
1. Создать аккаунт на sentry.io
2. Создать проект
3. Скопировать DSN
4. Добавить `SENTRY_DSN` в Vercel
