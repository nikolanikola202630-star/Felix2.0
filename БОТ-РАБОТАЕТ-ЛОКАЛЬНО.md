3. Или использовать другой AI API (OpenAI, Anthropic, etc.)

## 🔧 Настройка

### Изменить AI API ключ

Отредактируй `bot-local-polling.js`:

```javascript
const GROQ_API_KEY = 'твой_новый_ключ';
```

### Использовать другой AI

Замени функцию `getAIResponse()` на другой API:

**OpenAI:**
```javascript
async function getAIResponse(prompt) {
  const response = await fetch('https://api.openai.com/v1/chat/completions', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${OPENAI_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      model: 'gpt-3.5-turbo',
      messages: [{ role: 'user', content: prompt }]
    })
  });
  // ...
}
```

### Без AI (только команды)

Закомментируй AI вызовы:

```javascript
// Regular message - без AI
// const aiResponse = await getAIResponse(text);
await send(chatId, `Получил: ${text}`);
```
