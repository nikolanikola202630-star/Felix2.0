## API Endpoints

### Community API

#### GET /api/community/discussions
Получить обсуждения
```
?course_id=1          # ID курса или 'all'
&category=trading     # Категория
&limit=20            # Лимит
```

#### POST /api/community/discussions
Создать обсуждение
```json
{
  "user_id": 123456789,
  "course_id": 1,
  "category": "trading",
  "title": "Заголовок",
  "content": "Текст обсуждения"
}
```

#### GET /api/community/discussions/:id/comments
Получить комментарии

#### POST /api/community/discussions/:id/comments
Добавить комментарий
```json
{
  "user_id": 123456789,
  "content": "Текст комментария"
}
```

#### POST /api/community/discussions/:id/like
Лайкнуть/убрать лайк
```json
{
  "user_id": 123456789
}
```

#### GET /api/community/stats
Статистика сообщества
```json
{
  "members": 1234,
  "discussions": 567,
  "active": 89
}
```

### Settings API

#### GET /api/settings?user_id=123
Получить настройки пользователя

#### POST /api/settings
Сохранить настройки
```json
{
  "user_id": 123456789,
  "settings": {
    "theme": "dark",
    "ai-model": "llama-3.3-70b-versatile",
    "ai-temperature": 0.7,
    "notifications-lessons": true,
    ...
  }
}
```

## База данных

### Новые таблицы

#### discussions
```sql
id, user_id, course_id, category, 
title, content, is_pinned, is_locked,
created_at, updated_at
```

#### discussion_comments
```sql
id, discussion_id, user_id, 
content, created_at, updated_at
```

#### discussion_likes
```sql
id, discussion_id, user_id, created_at
UNIQUE(discussion_id, user_id)
```

### Обновленные таблицы

#### user_settings
```sql
+ preferences JSONB  -- Полные настройки персонализации
```

## Как работает персонализация

### 1. Локальное хранение
```javascript
localStorage.setItem('felixSettings', JSON.stringify(settings));
localStorage.setItem('felixPersonalization', JSON.stringify(personalization));
```

### 2. Синхронизация с сервером
```javascript
POST /api/settings
{
  user_id: 123456789,
  settings: { ... }
}
```

### 3. Глобальное применение
```javascript
// Загружается на всех страницах
<script src="js/personalization.js"></script>

// Применяет настройки:
- Тему (CSS variables)
- Анимации (CSS classes)
- Размер шрифта (CSS variables)
```

### 4. API для других скриптов
```javascript
// Получить настройку
window.FelixSettings.get('theme');

// Получить все настройки
window.FelixSettings.getAll();

// Получить персонализацию
window.FelixSettings.getPersonalization();
```
