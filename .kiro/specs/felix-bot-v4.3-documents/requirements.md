# Felix Bot v4.3 - Генерация документов

## Цель
Превратить Felix Bot в мощный инструмент для создания профессиональных документов с экспортом в PDF, DOCX и другие форматы.

## Основные возможности

### 1. Генерация академических работ
- **Рефераты** (5-15 страниц)
- **Курсовые работы** (20-40 страниц)
- **Дипломные работы** (50-100 страниц)
- **Эссе и статьи** (3-10 страниц)
- **Презентации** (10-30 слайдов)

### 2. Юридические документы
- **Договоры** (купли-продажи, аренды, подряда, оказания услуг)
- **Доверенности**
- **Заявления** (в суд, в госорганы)
- **Претензии и иски**
- **Соглашения** (о разделе имущества, алиментах)

### 3. Бизнес-документы
- **Бизнес-планы** (10-50 страниц)
- **Коммерческие предложения**
- **Отчеты** (финансовые, аналитические)
- **Презентации для инвесторов**
- **Маркетинговые материалы**

### 4. Личные документы
- **Резюме/CV** (1-2 страницы)
- **Сопроводительные письма**
- **Мотивационные письма**
- **Портфолио**
- **Биографии**

## Форматы экспорта

### PDF (приоритет)
- Профессиональное форматирование
- Оглавление с гиперссылками
- Нумерация страниц
- Колонтитулы
- Водяные знаки (опционально)

### DOCX
- Редактируемый формат
- Стили и форматирование
- Таблицы и изображения
- Совместимость с MS Word

### TXT
- Простой текст
- Для быстрого просмотра

### HTML
- Веб-версия документа
- Для публикации онлайн

## Технические требования

### Библиотеки для генерации
- **PDF**: `pdfkit` или `puppeteer`
- **DOCX**: `docx` или `officegen`
- **Markdown**: `marked` + `pdf-lib`
- **HTML**: встроенный

### Хранилище файлов
- **Vercel Blob Storage** (рекомендуется)
- **Supabase Storage** (альтернатива)
- **AWS S3** (для больших объемов)

### База данных
Таблица `documents`:
```sql
CREATE TABLE documents (
    id UUID PRIMARY KEY,
    user_id BIGINT NOT NULL,
    type VARCHAR(50) NOT NULL,
    title TEXT NOT NULL,
    content TEXT NOT NULL,
    format VARCHAR(10) NOT NULL,
    file_url TEXT,
    file_size INTEGER,
    pages INTEGER,
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW(),
    metadata JSONB
);
```

### Лимиты

  - Максимум 20 страниц
  - Только PDF и TXT
  
- **PRO план** (будущее):
  - Безлимитные документы
  - До 100 страниц
  - Все форматы
  - Приоритетная генерация

## Процесс генерации

### 1. Пользователь отправляет команду
```
/referat Искусственный интеллект в медицине
```

### 2. Бот собирает информацию
```
📝 Создаю реферат на тему "Искусственный интеллект в медицине"

Выберите параметры:
1️⃣ Объем: 10 / 15 / 20 страниц
2️⃣ Стиль: Научный / Популярный / Академический
3️⃣ Формат: PDF / DOCX / TXT

Или отправьте "Стандартно" для настроек по умолчанию
```

### 3. Генерация структуры
```
⏳ Генерирую структуру...

✅ Структура готова:
1. Введение
2. История развития ИИ в медицине
3. Современные применения
4. Диагностика заболеваний
5. Персонализированная медицина
6. Этические вопросы
7. Заключение
8. Список литературы

Продолжить? (Да/Изменить структуру)
```

### 4. Генерация контента
```
⏳ Генерирую контент... (это может занять 2-3 минуты)

📊 Прогресс:
▓▓▓▓▓▓▓▓░░ 80% (8/10 разделов)

Раздел "Этические вопросы" - готов ✅
```

### 5. Форматирование и экспорт
```
⏳ Форматирую документ...
⏳ Создаю PDF файл...

✅ Документ готов!

📄 Реферат: "Искусственный интеллект в медицине"
📊 Страниц: 15
📦 Размер: 2.3 MB
🕐 Время создания: 2 мин 34 сек

[📥 Скачать PDF] [📝 Редактировать] [🔄 Пересоздать]
```

## Интеграции

### AI для контента
- **Groq LLaMA 3.3 70B** - основная генерация
- **Claude/GPT** - для сложных документов (опционально)

### Проверка качества
- **Грамматика**: встроенная проверка
- **Плагиат**: интеграция с API (опционально)
- **Уникальность**: базовая проверка

### Оформление
- **Шрифты**: Times New Roman, Arial, Calibri
- **Размеры**: 12pt, 14pt
- **Интервалы**: 1.0, 1.5, 2.0
- **Поля**: стандартные, узкие, широкие

## Метрики успеха

- **Количество созданных документов** в день
- **Средний размер документа** (страниц)
- **Популярные типы документов**

- **Copy.ai** - только короткие тексты
