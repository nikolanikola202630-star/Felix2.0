## 🚀 Процесс деплоя

### Шаг 1: Финальная проверка

```bash
npm run deploy:check
```

Должно быть: ✅ Проект полностью готов к деплою!

### Шаг 2: Коммит изменений

```bash
git add .
git commit -m "feat: Felix Academy V12 - Ready for production

- Unified Mini App with brandbook integration
- Secure Telegram authentication
- Centralized error handling
- SPA router with navigation
- API V12 with validation
- Complete documentation

Old Money. Cold Mind. High Society."
git push
```

### Шаг 3: Деплой на Vercel

```bash
# Первый раз - связать проект
vercel link

# Деплой
vercel --prod
```

Сохраните URL: `https://your-app.vercel.app`

### Шаг 4: Настроить переменные в Vercel

```bash
# Через CLI
vercel env add TELEGRAM_BOT_TOKEN
vercel env add SUPABASE_URL
vercel env add SUPABASE_KEY
vercel env add GROQ_API_KEY
vercel env add ADMIN_ID
vercel env add NODE_ENV

# Или через Dashboard
# https://vercel.com/dashboard → Settings → Environment Variables
```

### Шаг 5: Обновить MINIAPP_URL

```bash
# В Vercel Dashboard
# Settings → Environment Variables
# MINIAPP_URL = https://your-app.vercel.app/miniapp/app.html

# Redeploy
vercel --prod
```

### Шаг 6: Установить webhook

```bash
npm run webhook:set
```

Или вручную:
```bash
curl -X POST "https://api.telegram.org/bot<TOKEN>/setWebhook" \
  -d "url=https://your-app.vercel.app/api/webhook"
```

### Шаг 7: Проверить webhook

```bash
curl "https://api.telegram.org/bot<TOKEN>/getWebhookInfo"
```

Должно быть:
```json
{
  "ok": true,
  "result": {
    "url": "https://your-app.vercel.app/api/webhook",
    "pending_update_count": 0
  }
}
```

### Шаг 8: Загрузить тестовые курсы

```bash
npm run upload:courses
```

### Шаг 9: Финальная проверка

1. Откройте бота в Telegram
2. Отправьте `/start`
3. Нажмите "Открыть приложение"
4. Проверьте все функции

## 🐛 Troubleshooting

### Проблема: Бот не отвечает

**Проверить:**
```bash
# Webhook
curl "https://api.telegram.org/bot<TOKEN>/getWebhookInfo"

# Логи
vercel logs

# Переменные
vercel env ls
```

**Решение:**
1. Переустановить webhook: `npm run webhook:set`
2. Проверить `TELEGRAM_BOT_TOKEN`
3. Проверить логи на ошибки

### Проблема: Приложение не открывается

**Проверить:**
```bash
# URL доступен
curl https://your-app.vercel.app/miniapp/app.html

# Переменные
vercel env ls | grep MINIAPP_URL
```

**Решение:**
1. Проверить `MINIAPP_URL` в переменных
2. Убедиться, что файл существует
3. Проверить логи браузера (F12)

### Проблема: Курсы не загружаются

**Проверить:**
```bash
# API
curl https://your-app.vercel.app/api/app?action=getCourses

# Supabase
npm run test:supabase
```

**Решение:**
1. Проверить `SUPABASE_URL` и `SUPABASE_KEY`
2. Убедиться, что таблица courses существует
3. Загрузить тестовые курсы: `npm run upload:courses`

### Проблема: Ошибки аутентификации

**Проверить:**
```bash
# Консоль браузера (F12)
# Должен быть initData от Telegram
```

**Решение:**
1. Убедиться, что приложение открыто через Telegram
2. Проверить `TELEGRAM_BOT_TOKEN`
3. Очистить кэш браузера
