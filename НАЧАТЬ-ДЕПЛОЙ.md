## 🚀 Следующие шаги

### Шаг 1: Пуш в GitHub (1 мин)

```bash
git push origin main
```

### Шаг 2: База данных (10 мин)

1. Открыть https://supabase.com
2. Создать новый проект
3. SQL Editor → Выполнить `database/migrations/001-add-ml-tables-safe.sql`
4. Скопировать DATABASE_URL

### Шаг 3: API ключи (5 мин)

1. **Telegram:** @BotFather → `/newbot`
2. **Groq:** https://console.groq.com → API Keys
3. **Admin ID:** @userinfobot

### Шаг 4: Vercel (10 мин)

1. https://vercel.com → New Project
2. Import репозиторий
3. Добавить Environment Variables:
   - TELEGRAM_BOT_TOKEN
   - GROQ_API_KEY
   - DATABASE_URL
   - ADMIN_ID
   - MINIAPP_URL
   - NODE_ENV=production
4. Deploy

### Шаг 5: Webhook (5 мин)

```bash
curl -X POST "https://api.telegram.org/bot<TOKEN>/setWebhook" \
  -d "url=https://your-project.vercel.app/api/webhook"
```

### Шаг 6: Тестирование (5 мин)

```bash
# Health check
curl https://your-project.vercel.app/api/webhook

# Telegram: /start, /help, /profile
# Mini App: открыть через кнопку
```
