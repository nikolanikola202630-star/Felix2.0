## 🚀 ЧТО ДЕЛАТЬ ПРЯМО СЕЙЧАС

### 1. Push в GitHub (1 минута)

```powershell
# GitHub Desktop уже открыт
# Просто нажмите "Push origin"
```

Или через командную строку:
```powershell
git push origin main
```

### 2. Запустить Vercel Setup (3 минуты)

```powershell
.\vercel-quick-setup.ps1
```

Скрипт:
- ✅ Проверит всё автоматически
- ✅ Покажет что нужно настроить
- ✅ Откроет Dashboard
- ✅ Предложит деплой

### 3. Настроить Environment Variables (5 минут)

**Vercel Dashboard → Settings → Environment Variables**

Добавить:
- `SENTRY_DSN` (создать на sentry.io)
- `AI_DAILY_LIMIT=50`
- `AI_HOURLY_LIMIT=10`
- `KV_URL` (создать Vercel KV)
- `KV_REST_API_URL`
- `KV_REST_API_TOKEN`
- `KV_REST_API_READ_ONLY_TOKEN`

### 4. Деплой (1 минута)

```powershell
vercel --prod
```
