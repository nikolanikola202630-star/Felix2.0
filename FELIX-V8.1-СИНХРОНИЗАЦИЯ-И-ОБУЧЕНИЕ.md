# Felix v8.1 - Синхронизация и Адаптивное Обучение

## 🎉 ЧТО ДОБАВЛЕНО

### 1. Система Синхронизации (User Sync)
**Файл:** `lib/sync/user-sync.js`

Синхронизация данных между ботом и Mini App в реальном времени:
- Автоматическая синхронизация статистики пользователя
- Кэширование данных для быстрого доступа
- Отслеживание активности (сообщения, AI запросы, уроки)
- Система наград (XP за действия)
- Синхронизация настроек между платформами

**Возможности:**
```javascript
// Синхронизация данных пользователя
await userSync.syncUserData(userId);

// Обновление статистики
await userSync.updateStats(userId, { xp: 100, level: 5 });

// Отслеживание активности
await userSync.trackActivity(userId, { type: 'ai_request' });
```

### 2. Адаптивное Обучение (Adaptive Learning)
**Файл:** `lib/learning/adaptive-learning.js`

AI-система которая адаптируется под каждого пользователя:
- Анализ стиля обучения (визуальный/аудиальный/кинестетический)
- Определение темпа обучения (медленный/средний/быстрый)
- Выявление сильных и слабых сторон
- Генерация персонализированного контента
- Рекомендации следующих уроков
- Отслеживание прогресса

**Возможности:**
```javascript
// Анализ стиля обучения
const profile = await adaptiveLearning.analyzeLearningStyle(userId, interactions);

// Генерация персонализированного контента
const content = await adaptiveLearning.generatePersonalizedContent(userId, 'JavaScript', 'lesson');

// Рекомендация следующего урока
const recommendation = await adaptiveLearning.recommendNextLesson(userId, completedLessons);

// Отслеживание прогресса
await adaptiveLearning.trackProgress(userId, lessonId, score, timeSpent);
```

### 3. API Синхронизации
**Файл:** `api/sync.js`
**Endpoint:** `/api/sync`

REST API для взаимодействия Mini App с системами синхронизации и обучения:

**Действия:**
- `getUserData` - Получить все данные пользователя
- `updateStats` - Обновить статистику
- `trackActivity` - Отследить активность
- `getLearningProfile` - Получить профиль обучения
- `getRecommendation` - Получить рекомендацию урока
- `trackProgress` - Отследить прогресс
- `generateContent` - Сгенерировать персонализированный контент

**Пример использования:**
```javascript
// Получить данные пользователя
const response = await fetch('/api/sync', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    action: 'getUserData',
    userId: 123456
  })
});
```

## 🎯 КАК ЭТО РАБОТАЕТ

### Сценарий 1: Пользователь начинает обучение

1. Пользователь открывает курс в Mini App
2. Система отслеживает активность через `/api/sync`
3. AI анализирует стиль обучения пользователя
4. Генерируется персонализированный контент
5. Прогресс синхронизируется с ботом
6. Пользователь получает XP и достижения

### Сценарий 2: Пользователь общается с ботом

1. Пользователь отправляет сообщение боту
2. Бот проверяет настройки пользователя (модель AI, температура)
3. Генерирует ответ с учетом предпочтений
4. Отслеживает активность (+5 XP за сообщение)
5. Раз в день анализирует стиль общения
6. Адаптирует ответы под пользователя

### Сценарий 3: Персонализированные рекомендации

1. Система анализирует пройденные уроки
2. Определяет сильные и слабые стороны
3. AI генерирует рекомендацию следующего урока
4. Учитывает темп обучения пользователя
5. Предлагает оптимальный контент
