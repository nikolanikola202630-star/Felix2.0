# ⚡ Быстрый переход на бесплатный режим

## 🎯 Цель
Сделать все курсы бесплатными за 5 минут.

---

## 📋 Чеклист (5 шагов)

### 1. Применить миграцию БД (1 мин)

```sql
-- Подключиться к Supabase и выполнить:
ALTER TABLE courses ADD COLUMN IF NOT EXISTS is_free BOOLEAN DEFAULT true;
UPDATE courses SET is_free = true;
```

### 2. Проверить изменения (1 мин)

```bash
# Запустить скрипт проверки
node scripts/convert-to-free.js
```

Должно показать:
```
✅ api/payments.js - disabled
✅ api/courses/check-access.js - updated
✅ miniapp/js/catalog.js - updated
```

### 3. Деплой (2 мин)

```bash
git add .
git commit -m "feat: бесплатный доступ ко всем курсам (V13.0)"
git push
```

Vercel автоматически задеплоит изменения.

### 4. Проверить в Mini App (1 мин)

1. Открыть https://felix2-0.vercel.app/miniapp/catalog.html
2. Проверить: нет цен, есть "Бесплатно"
3. Открыть любой курс
4. Убедиться: все уроки доступны

### 5. Готово! ✅

Все курсы теперь бесплатны!

## 🔄 Как вернуть платные функции

1. Раскомментировать код с `/* PAYMENT:`
2. Выполнить: `UPDATE courses SET is_free = false WHERE price > 0;`
3. Деплой
