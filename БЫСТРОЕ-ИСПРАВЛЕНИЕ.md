## 🔧 Если что-то не работает

### Проблема: Ошибка в БД

**Решение:**
```sql
-- Проверить, какие таблицы есть
SELECT table_name FROM information_schema.tables 
WHERE table_schema = 'public' 
ORDER BY table_name;

-- Если нет нужных таблиц - запустить миграцию еще раз
```

### Проблема: Бот не отвечает

**Решение:**
```bash
# Проверить логи
vercel logs --follow

# Проверить webhook
curl https://api.telegram.org/bot<YOUR_TOKEN>/getWebhookInfo
```

### Проблема: Ошибки в коде

**Решение:**
```bash
# Откатить изменения
git revert HEAD
git push

# Проверить еще раз файлы
```
