# 🚀 Ручной деплой через Vercel Dashboard

**Проблема:** Достигнут лимит деплоев (100/день)  
**Решение:** Запустить деплой вручную через веб-интерфейс  
**Время:** 3 минуты

---

## ⚡ Быстрая инструкция

### Шаг 1: Открыть Vercel Dashboard
```
https://vercel.com/dashboard
```

### Шаг 2: Выбрать проект
- Найти проект **Felix2.0** (или ваше название)
- Кликнуть на него

### Шаг 3: Перейти в Deployments
- В верхнем меню выбрать **Deployments**
- Увидите список всех деплоев

### Шаг 4: Найти последний коммит
Найти коммит:
```
93b7e35 - Felix Academy V12 + Brandbook: Old Money. Cold Mind. High Society.
```

### Шаг 5: Запустить Redeploy
- Нажать на три точки (...) справа от коммита
- Выбрать **Redeploy**
- Подтвердить действие

### Шаг 6: Дождаться завершения
- Деплой займёт ~2-3 минуты
- Статус изменится на **Ready**
- Получите ссылку на сайт

---

## 🎯 Альтернативный способ

### Через Git (автоматический деплой)
Vercel автоматически деплоит при push в main:

```bash
# Сделать любое небольшое изменение
echo "# Deploy trigger" >> README.md

# Закоммитить
git add README.md
git commit -m "Trigger deploy"
git push
```

**Но:** Это сработает только если лимит уже обновился (через 5 часов).

---

## ⏰ Когда обновится лимит?

Лимит обновляется каждые 24 часа с момента первого деплоя дня.

**Текущий статус:**
- Лимит: 100 деплоев/день
- Использовано: 100
- Осталось: 0
- Обновление: через ~5 часов
