Vercel автоматически задеплоит за 1-2 минуты.

---

## Шаг 3: Обновить переменную в Vercel (2 минуты)

1. Откройте: https://vercel.com/dashboard
2. Выберите проект: `felix2-0`
3. Settings → Environment Variables
4. Найдите: `MINIAPP_URL`
5. Измените на: `https://felix2-0.vercel.app/miniapp/egoist.html`
6. Save
7. Deployments → Redeploy (последний деплой)

---

## Шаг 4: Проверка (1 минута)

## 🐛 Если не работает

### Проблема: Бот открывает старую версию

**Решение:**
```bash
# Очистить кэш Telegram
node scripts/clear-telegram-cache.js

# Или подождать 5-10 минут
```

### Проблема: 404 на egoist.html

**Решение:**
```bash
# Проверить, что файл есть
git ls-files | grep egoist.html

# Если нет, добавить
git add miniapp/egoist.html
git commit -m "add: egoist.html"
git push
```

### Проблема: Дизайн не применился

**Решение:**
- Очистить кэш браузера (Ctrl+Shift+R)
- Проверить, что CSS загружен
- Подождать 2-3 минуты
