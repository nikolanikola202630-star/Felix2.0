## Команды бота

### Основные
- `/start` - Начать работу
- `/help` - Помощь
- `/profile` - Профиль пользователя
- `/stats [period]` - Статистика (day/week/month/all)
- `/history [limit]` - История диалогов
- `/search [query]` - Поиск по сообщениям

### AI команды
- `/ask [текст]` - Задать вопрос
- `/summary [текст]` - Краткое содержание
- `/analyze [текст]` - Анализ текста
- `/generate [тема]` - Генерация контента
- `/translate [текст]` - Перевод на английский
- `/improve [текст]` - Улучшить текст
- `/brainstorm [тема]` - Генерация идей
- `/explain [тема]` - Объяснение

### Админ
- `/admin` - Админ панель (только для ADMIN_ID)

### Группы
- `/groupstats` - Статистика группы
