## 🎯 Задача: Подключить Supabase

### Шаг 1: Установить зависимости (1 мин)

```bash
npm install
```

Это установит:
- `@supabase/supabase-js` - клиент Supabase
- `dotenv` - переменные окружения
- Все остальные зависимости

---

### Шаг 2: Получить credentials (2 мин)

1. Открыть https://supabase.com/dashboard
2. Выбрать проект Felix Academy
3. Settings → API
4. Скопировать:
   - **Project URL** → это будет `SUPABASE_URL`
   - **anon public** ключ → это будет `SUPABASE_KEY`

## 🚀 Что дальше?

### Следующий шаг: Добавить в Vercel (5 мин)

```bash
# Добавить переменные
vercel env add SUPABASE_URL production
# Вставить: https://your-project.supabase.co

vercel env add SUPABASE_KEY production
# Вставить: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

# Задеплоить
vercel --prod
```

### Проверить на production (1 мин)

```bash
curl https://felix2-0.vercel.app/api/health/database
```

Должно вернуть:
```json
{
  "success": true,
  "database": {
    "connected": true,
    "message": "Database connection OK"
  },
  "status": "healthy"
}
```

## 🆘 Если что-то не работает

### Проблема: "Supabase not configured"

**Решение:**
1. Проверить, что `.env` содержит правильные значения
2. Проверить, что `npm install` выполнен
3. Перезапустить скрипт

### Проблема: "Database connection failed"

**Решение:**
1. Проверить URL (должен быть `https://xxxxx.supabase.co`)
2. Проверить ключ (должен начинаться с `eyJ`)
3. Проверить статус проекта в Supabase Dashboard

### Проблема: "Function not found"

**Решение:**
1. Применить SQL функцию из `database/functions/update_bonus_balance.sql`
2. Проверить в Supabase Dashboard → Database → Functions
