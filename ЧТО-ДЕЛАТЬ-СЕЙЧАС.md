## ⚠️ Если что-то не работает

### Проблема: API возвращает ошибку

**Решение:**
```bash
# Проверить логи
vercel logs --follow

# Проверить переменные окружения
vercel env ls
```

### Проблема: Бот не отвечает

**Решение:**
1. Проверить webhook:
   ```bash
   curl https://api.telegram.org/bot<YOUR_TOKEN>/getWebhookInfo
   ```
2. Должен быть URL: `https://felix2-0.vercel.app/api/webhook`
3. Должен быть `pending_update_count: 0`

### Проблема: Ошибки в /profile или /stats

**Решение:**
1. Проверить таблицы в Supabase:
   ```sql
   SELECT table_name FROM information_schema.tables 
   WHERE table_schema = 'public' ORDER BY table_name;
   ```
2. Должно быть 22 таблицы
3. Если нет - запустить `database/FULL-SCHEMA-COMBINED.sql` еще раз

### Проблема: Mini App не открывается

**Решение:**
1. Проверить файл существует:
   ```bash
   curl https://felix2-0.vercel.app/miniapp/elite-v6.html
   ```
2. Должен вернуть HTML код
