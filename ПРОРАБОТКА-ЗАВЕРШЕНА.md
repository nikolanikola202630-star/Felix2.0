## 🎯 Что проработано

### 1. Полное управление через админ панель

**Создан расширенный API:** `api/admin/courses-manage.js`

**Новые функции:**

✅ **Управление ценами**
- `updatePrice` - изменить цену курса
- `bulkUpdatePrices` - массовое изменение цен со скидкой
- Автоматический расчет в Stars (1 Star = 2₽)

✅ **Управление уроками**
- `addLesson` - добавить урок к теме
- `updateLesson` - редактировать урок
- `deleteLesson` - удалить урок
- `reorderLessons` - изменить порядок уроков

✅ **Детальная статистика**
- `getDetailedStats` - полная аналитика
- По категориям, уровням, ценам
- Средние показатели
- Список всех курсов с метриками

### 2. Система покупок и комиссий

**Создан модуль:** `lib/db-purchases.js`

**Функции:**

✅ **Покупки**
- `createPurchase` - сохранение покупки в БД
- `checkPurchase` - проверка доступа к курсу
- `getUserPurchases` - все покупки пользователя
- `refundPurchase` - возврат средств

✅ **Комиссии**
- `processCommissions` - автоматическое начисление
- 20% реферерам
- Запись в bonus_transactions
- Обновление баланса

✅ **Статистика**
- `getPurchaseStats` - аналитика продаж
- По курсам и валютам
- Общая выручка

### 3. Проверка доступа к курсам

**Создан API:** `api/courses/check-access.js`

**Возможности:**
- Проверка покупки курса
- Список доступных уроков
- Бесплатные уроки (preview)
- Статистика доступа

### 4. Интеграция с платежами

**Обновлен:** `api/payments/webhook.js`

**Улучшения:**
- Использует `lib/db-purchases.js`
- Автоматическое сохранение в БД
- Начисление комиссий
- Уведомления пользователям

## 🎨 UI Админ панели

### Управление ценами

```html
<div class="price-editor">
  <h3>💰 Управление ценами</h3>
  
  <div class="course-price">
    <label>Курс: Детективная лаборатория сознания</label>
    <div class="price-input">
      <input type="number" value="2990" id="price" />
      <span class="currency">₽</span>
      <span class="stars-preview">~1495 Stars</span>
    </div>
    <button onclick="updatePrice(1)">Сохранить</button>
  </div>
  
  <div class="bulk-discount">
    <h4>Массовая скидка</h4>
    <label>Выберите курсы:</label>
    <div class="course-checkboxes">
      <label><input type="checkbox" value="1" /> Психология</label>
      <label><input type="checkbox" value="2" /> Саморазвитие</label>
      <label><input type="checkbox" value="3" /> Бизнес</label>
    </div>
    <label>Скидка (%):</label>
    <input type="number" value="20" id="discount" />
    <button onclick="applyBulkDiscount()">Применить</button>
  </div>
</div>

<script>
async function updatePrice(courseId) {
  const price = document.getElementById('price').value;
  
  const response = await fetch('/api/admin/courses-manage', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      action: 'updatePrice',
      userId: tg.initDataUnsafe.user.id,
      courseId,
      price: parseInt(price)
    })
  });
  
  const data = await response.json();
  
  if (data.success) {
    tg.showAlert(`✅ Цена обновлена: ${data.course.new_price}₽`);
  } else {
    tg.showAlert(`❌ Ошибка: ${data.error}`);
  }
}

async function applyBulkDiscount() {
  const checkboxes = document.querySelectorAll('.course-checkboxes input:checked');
  const courseIds = Array.from(checkboxes).map(cb => parseInt(cb.value));
  const discount = parseInt(document.getElementById('discount').value);
  
  if (courseIds.length === 0) {
    tg.showAlert('Выберите хотя бы один курс');
    return;
  }
  
  const response = await fetch('/api/admin/courses-manage', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      action: 'bulkUpdatePrices',
      userId: tg.initDataUnsafe.user.id,
      courseIds,
      discount
    })
  });
  
  const data = await response.json();
  
  if (data.success) {
    tg.showAlert(`✅ Обновлено ${data.updated_count} курсов`);
    location.reload();
  } else {
    tg.showAlert(`❌ Ошибка: ${data.error}`);
  }
}
</script>
```
