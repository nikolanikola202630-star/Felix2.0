# 🤖 Felix Bot v4.2 - AI Ассистент

Умный Telegram бот с расширенными AI командами, голосовым вводом и красивым Mini App интерфейсом.

## ✨ Новые возможности v4.2

### 🌍 Перевод текста
- Команда `/translate <язык> <текст>` для перевода
- Поддержка 12+ языков (en, ru, es, fr, de, it, pt, zh, ja, ko, ar, hi)
- Сохранение стиля и смысла оригинала

### ✨ Улучшение текста
- Команда `/improve <текст>` для улучшения
- Исправление ошибок
- Повышение читаемости и профессионализма

### 💡 Генерация идей
- Команда `/brainstorm <тема>` для мозгового штурма
- 5-7 креативных идей с описанием
- Разнообразные подходы к теме

### 📚 Простое объяснение
- Команда `/explain <текст>` для объяснения
- Простой язык, понятный каждому
- Аналогии и примеры из жизни

### 📊 Статистика использования
- Команда `/stats` для просмотра статистики
- Количество использованных команд
- Дата первого и последнего использования
- Топ команд

### 🎤 Голосовой ввод
- Распознавание голосовых сообщений через Groq Whisper Large v3
- Поддержка русского языка
- Автоматическая обработка и ответ AI

### 💬 AI Диалоги
- Ответы на вопросы через LLaMA 3.3 70B
- Контекстная память (последние 10 сообщений)
- Естественное общение на русском языке

### 📝 Организация текста
- Команда `/organize <текст>` для структурирования
- Автоматическое создание списков и категорий
- Выделение ключевых моментов

### 📱 Mini App
- Красивый интерфейс с градиентным дизайном
- Демонстрация всех возможностей v4.2
- Примеры использования команд
- Технологический стек

## 📋 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню с кнопкой Mini App |
| `/translate <язык> <текст>` | Перевод текста на другой язык |
| `/improve <текст>` | Улучшение текста |
| `/brainstorm <тема>` | Генерация идей по теме |
| `/explain <текст>` | Простое объяснение сложного |
| `/stats` | Статистика использования |
| `/organize <текст>` | Структурировать текст |
| `/clear` | Очистить историю диалогов |
| Голосовое | Отправьте голосовое - оно будет распознано |
| Текст | Любое сообщение - получите AI ответ с контекстом |

## 🏗️ Архитектура

```
Telegram Bot (@fel12x_bot)
    ↓
Vercel Serverless Functions
    ├── api/webhook.js (основной обработчик + in-memory storage)
    └── miniapp/index.html (Mini App)
    ↓
External Services
    └── Groq API (LLaMA 3.3 70B + Whisper v3)
```

## 🛠️ Технологии

- **Backend**: Node.js (Vercel Serverless)
- **AI**: Groq API
  - LLaMA 3.3 70B Versatile (текстовая генерация)
  - Whisper Large v3 (распознавание речи)
- **Hosting**: Vercel
- **Bot API**: Telegram Bot API
- **Frontend**: Vanilla JS + Telegram WebApp API

## 📝 Примеры использования

### Перевод текста
```
Пользователь: /translate en Привет, как дела?
Felix: 🌍 Перевод на Английский:

Hello, how are you?
```

### Улучшение текста
```
Пользователь: /improve я хочу сказать что этот продукт очень хороший и мне нравится
Felix: ✨ Улучшенный текст:

Хочу отметить, что этот продукт превосходного качества и полностью соответствует моим ожиданиям.
```

### Генерация идей
```
Пользователь: /brainstorm идеи для YouTube канала о программировании
Felix: 💡 Идеи по теме "идеи для YouTube канала о программировании":

1. "Код с нуля" - обучение программированию для начинающих
2. "Разбор кода" - анализ популярных open-source проектов
3. "Челленджи" - решение задач на время
4. "Интервью" - беседы с опытными разработчиками
5. "Лайфхаки" - полезные советы и трюки
```

### Простое объяснение
```
Пользователь: /explain что такое блокчейн
Felix: 📚 Простое объяснение:

Представь блокчейн как цепочку блоков с информацией, где каждый блок связан с предыдущим. 
Это как дневник, который нельзя изменить - можно только добавлять новые записи. 
Все участники имеют копию этого дневника, поэтому обмануть систему невозможно.
```

### Статистика

Felix: 📊 Ваша статистика:

🕐 Последнее использование: 01.03.2026

Команды:

  • improve: 10
