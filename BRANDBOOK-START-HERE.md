## ⚡ Быстрый старт (5 минут)

### Шаг 1: Посмотрите демо

Откройте в браузере:
```
miniapp/brandbook-demo.html
```

Вы увидите все компоненты брендбука в действии.

### Шаг 2: Запустите интеграцию

```bash
node scripts/integrate-brandbook.js
```

Скрипт автоматически:
- Создаст резервные копии
- Заменит старые файлы новыми
- Обновит ссылки
- Сгенерирует отчёт

### Шаг 3: Проверьте результат

Откройте в браузере:
```
miniapp/elite.html
```

Готово! Брендбук интегрирован.

## 🎨 Основные принципы

### Цвета
- **Антрацит** (#1A1C1E) - основной фон
- **Золото** (#C4A962) - акценты (≤ 10%)
- **Изумруд** (#1E4D3E) - успех
- **Мрамор** (#F5F5F3) - текст

### Типографика
- **Cormorant Garamond** - заголовки
- **Montserrat** - основной текст
- **Playfair Display** - акцентные числа

### Tone of Voice
- Обращение на "Вы"
- Предложения до 15 слов
- Без смайликов (кроме 🦁 ⚜️ 📜)
- Без восклицательных знаков
