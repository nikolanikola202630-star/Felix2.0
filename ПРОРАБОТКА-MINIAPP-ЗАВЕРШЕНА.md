## Ключевые особенности

### 1. Единый стиль
- Все страницы используют `flagship-premium.css`
- Консистентные цвета и отступы через CSS-переменные
- Плавные анимации и переходы

### 2. Telegram WebApp интеграция
- BackButton для навигации
- HapticFeedback для тактильной обратной связи
- Expand для полноэкранного режима
- Адаптация под тему Telegram

### 3. Производительность
- Debounce для поиска
- Lazy loading изображений
- LocalStorage для кэширования
- Fallback на локальные данные

### 4. UX/UI
- Stagger-анимации для списков
- Empty states для пустых состояний
- Loading states для загрузки
- Понятные иконки и эмодзи

## API Endpoints (используются)

```javascript
GET /api/courses          // Список курсов
GET /api/courses/:slug    // Детали курса
GET /api/user/progress    // Прогресс пользователя
GET /api/user/settings    // Настройки пользователя
POST /api/user/settings   // Сохранение настроек
```

## LocalStorage ключи

```javascript
'felixSettings'      // Настройки приложения
'felixUser'          // Данные пользователя
'likedPosts'         // Лайкнутые посты
'voiceHistory'       // История голосовых записей
'coursesCache'       // Кэш курсов
```
