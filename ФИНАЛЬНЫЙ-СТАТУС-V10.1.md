## 📊 API Endpoints

### Папки:
- `GET /api/ai-chat-folders?userId=123` - получить все папки
- `POST /api/ai-chat-folders?userId=123` - создать папку
- `PUT /api/ai-chat-folders/1?userId=123` - обновить папку
- `DELETE /api/ai-chat-folders/1?userId=123` - удалить папку

### Чаты:
- `GET /api/ai-chat-folders/chats?userId=123&folderId=1` - получить чаты
- `POST /api/ai-chat-folders/chats?userId=123` - создать чат
- `PUT /api/ai-chat-folders/chats/1?userId=123` - обновить чат
- `DELETE /api/ai-chat-folders/chats/1?userId=123` - удалить чат

### Сообщения:
- `GET /api/ai-chat-folders/chats/1/messages?userId=123` - получить сообщения
- `POST /api/ai-chat-folders/chats/1/messages?userId=123` - отправить сообщение

## 📱 Как использовать

### Для пользователей:

1. **Открыть AI чат**
   ```
   https://felix2-0.vercel.app/ai-chat.html
   ```

2. **Создать папку**
   ```javascript
   POST /api/ai-chat-folders?userId=123
   {
     "name": "Проекты",
     "icon": "🚀",
     "color": "#EF4444"
   }
   ```

3. **Создать чат**
   ```javascript
   POST /api/ai-chat-folders/chats?userId=123
   {
     "folder_id": 1,
     "title": "Мой проект",
     "model": "llama-3.3-70b"
   }
   ```

4. **Отправить сообщение**
   ```javascript
   POST /api/ai-chat-folders/chats/1/messages?userId=123
   {
     "content": "Привет, AI!"
   }
   ```
