# 📝 Changelog - Felix Bot v5.0

## ✨ Новые возможности

### 🧠 Система самообучения
- **Профили пользователей** - Автоматическое создание и обновление
- **Анализ стиля общения** - Определение формальности (formal/casual/mixed)
- **Отслеживание эмодзи** - Среднее количество на сообщение
- **Анализ длины сообщений** - Адаптация под предпочтения
- **Извлечение интересов** - Автоматическое определение тем
- **Персонализированные промпты** - AI адаптируется под каждого
- **Прогресс обучения** - 0-100% в зависимости от общения
- **Команда `/profile`** - Просмотр профиля обучения
- **Команда `/stats`** - Детальная статистика

### 🛡️ Администрирование групп
- **Автоматическая модерация** - Спам, CAPS, повторы
- **Система предупреждений** - 3 варна = автобан на 24 часа
- **Автоудаление нарушений** - Мгновенное удаление
- **Приветствие новых** - Автоматическое с правилами
- **Настройки группы** - Гибкая конфигурация
- **Статистика группы** - Сообщения, активные, топ участников
- **Команда `/groupstats`** - Статистика (только админы)
- **FAQ система** - Структура готова

### 📱 Mini App v5.0
- **4 вкладки** - Профиль, Команды, Группа, Настройки
- **Вкладка "Профиль"**:
  - Прогресс обучения с анимацией
  - 4 статистики (сообщения, команды, стиль, эмодзи)
  - Список интересов
  - Кнопка обновления
- **Вкладка "Команды"**:
  - Все 8 AI команд
  - Быстрый запуск (тап = отправка)
  - Группировка по категориям
- **Вкладка "Группа"**:
  - Статус модерации
  - Настройка приветствия
  - Статистика группы
  - FAQ управление
- **Вкладка "Настройки"**:
  - Переключатели самообучения
  - Настройки уведомлений
  - Информация о боте
- **API интеграция** - Реальные данные через `/api/miniapp`
- **Красивый дизайн** - Градиенты, анимации, адаптивность

### 🤖 AI Команды (обновлены)
- **`/ask [вопрос]`** - Теперь с персонализацией под пользователя
- **`/summary [текст]`** - Адаптивная длина ответа
- **`/analyze [текст]`** - Учитывает стиль пользователя
- **`/generate [тема]`** - Персонализированная генерация
- **`/translate [текст]`** - С учетом предпочтений
- **`/improve [текст]`** - Адаптация под стиль
- **`/brainstorm [тема]`** - Учитывает интересы
- **`/explain [концепция]`** - Подстраивается под уровень
