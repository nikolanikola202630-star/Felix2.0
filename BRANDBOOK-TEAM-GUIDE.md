## 🎨 Для дизайнеров

### Что читать?
1. **BRANDBOOK-V12.md** - полный брендбук с философией и правилами
2. **BRANDBOOK-ICONS.md** - система иконок

### Основные правила
- Цветовая палитра: антрацит, графит, мрамор, золото (макс 10%)
- Шрифты: Cormorant Garamond (заголовки), Montserrat (текст), Playfair Display (цифры)
- Линии: тонкие (1px), золотые для акцентов
- Радиусы: 0-4px для строгости
- Тени: глубокие, контрастные

### Инструменты
- **Figma:** используйте переменные из `brandbook-variables.css`
- **Цвета:** #1A1C1E, #2D2F31, #F5F5F3, #C4A962, #E5E4E2
- **Демо:** откройте `miniapp/brandbook-demo.html` для референса

### Чек-лист перед отправкой макета
- [ ] Золото занимает не более 10% экрана
- [ ] Использованы только утвержденные цвета
- [ ] Заголовки в Cormorant Garamond, uppercase
- [ ] Текст в Montserrat
- [ ] Линии тонкие (1px)
- [ ] Есть легкая шумовая текстура на фонах
- [ ] Иконки в линейном стиле

## 💻 Для разработчиков

### Что читать?
1. **BRANDBOOK-README.md** - быстрый старт (5 минут)
2. **BRANDBOOK-IMPLEMENTATION.md** - подробная инструкция по внедрению

### Быстрый старт

#### 1. Подключите стили

```html
<link href="https://fonts.googleapis.com/css2?family=Cormorant+Garamond:wght@400;600;700&family=Montserrat:wght@300;400;500;600;700&family=Playfair+Display:wght@700;800;900&display=swap" rel="stylesheet">
<link rel="stylesheet" href="styles/brandbook-variables.css">
<link rel="stylesheet" href="css/brandbook-components.css">
```

#### 2. Используйте готовые компоненты

```html
<!-- Приветствие -->
<div class="welcome-card-brandbook">
  <div class="welcome-content-brandbook">
    <div class="avatar-brandbook">👤</div>
    <div class="welcome-text-brandbook">
      <h2>Добро пожаловать, [Имя].</h2>
      <p>Мы вас ждали.</p>
    </div>
  </div>
</div>

<!-- Кнопка -->
<button class="btn-brandbook btn-brandbook-primary">
  Начать обучение
</button>
```

#### 3. Используйте CSS переменные

```css
background: var(--bg-primary);      /* Антрацит */
color: var(--text-primary);         /* Мрамор */
border: 1px solid var(--border-gold); /* Золотая граница */
```

### Готовые компоненты
- `.welcome-card-brandbook` - приветственная карточка
- `.stat-card-brandbook` - статистика
- `.card-brandbook` - универсальная карточка
- `.btn-brandbook` - кнопки
- `.nav-brandbook` - навигация
- `.modal-brandbook` - модальные окна

- [ ] Проверена адаптивность на мобильных
- [ ] Золото занимает не более 10%

## ✍️ Для копирайтеров

### Что читать?
1. **BRANDBOOK-VOICE-EXAMPLES.md** - примеры текстов и фраз

### Тон коммуникации

**Принципы:**
- Уважительный, но холодный
- Лаконичный и точный
- Интеллектуальный
- Обращение на "Вы" с большой буквы

### Примеры

#### ✅ Правильно
- "Добро пожаловать, [Имя]. Мы вас ждали."
- "Средства зачислены. Ваш баланс: 2,450₽."
- "Операция выполнена."
- "Запрос принят. Ожидайте."

#### ❌ Неправильно
- "Привет! Рады тебя видеть!"
- "Ура! Оплата прошла!"
- "Упс, что-то пошло не так"
- "Окей, делаем!"

### Правила написания
1. Обращение на "Вы" - всегда с большой буквы
2. Никаких смайликов - только символы: ⚜️, 🦁, 📜
3. Короткие предложения - максимум 15 слов
4. Точные цифры - "2,450₽", а не "примерно 2,5 тысячи"
5. Без восклицательных знаков - только точки
6. Без "воды" - каждое слово несет смысл
7. Формальный тон - как в деловой переписке

### Чек-лист перед публикацией текста
- [ ] Обращение на "Вы" с большой буквы
- [ ] Нет смайликов (только символы)
- [ ] Предложения короткие (до 15 слов)
- [ ] Нет восклицательных знаков
- [ ] Нет "воды" и лишних слов
- [ ] Тон формальный и уважительный
- [ ] Цифры точные, не округленные

## 🎯 Философия бренда

### Миссия
Предоставить эксклюзивное образовательное пространство для тех, кто стремится к интеллектуальному лидерству и финансовой независимости.

### Ценности
- **Превосходство** - только лучшее
- **Интеллект** - холодный расчет, стратегическое мышление
- **Элитарность** - закрытое сообщество, привилегии
- **Традиции** - классика + инновации
- **Сдержанность** - истинная роскошь не кричит

### Концепция
**"Old Money. Cold Mind. High Society."**

Три столпа, формирующие уникальный характер бренда.
