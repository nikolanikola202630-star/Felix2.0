## 🎓 Использование

### Для пользователей
1. Открой бота: @fel12x_bot
2. Нажми /start
3. Используй команды или просто пиши
4. Открой Mini App для полного функционала

### Для разработчиков
1. Clone репозиторий
2. Установи зависимости: `npm install`
3. Настрой .env файл
4. Запусти локально: `npm run dev`
5. Деплой: `npm run deploy`
