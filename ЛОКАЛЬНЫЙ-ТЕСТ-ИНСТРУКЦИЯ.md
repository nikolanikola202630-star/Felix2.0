## 🐛 Возможные проблемы:

### Проблема 1: Порт 3000 занят
```bash
# Найти процесс на порту 3000
netstat -ano | findstr :3000

# Остановить процесс (замените PID)
Stop-Process -Id <PID> -Force
```

### Проблема 2: Бот не отвечает (409 Conflict)
```bash
# Остановить все процессы node
Stop-Process -Name node -Force

# Запустить заново
node server-local.js  # Терминал 1
node bot.js           # Терминал 2
```

### Проблема 3: Mini App не открывается
- Проверьте, что сервер запущен (терминал 1)
- Проверьте URL в bot.js: `http://localhost:3000/miniapp/index.html`
- Откройте URL напрямую в браузере для проверки

### Проблема 4: AI не отвечает
- Проверьте API ключ Groq в bot.js
- Проверьте интернет-соединение
- Проверьте лимиты API
